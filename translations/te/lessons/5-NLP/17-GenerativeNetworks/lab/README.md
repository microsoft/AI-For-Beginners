# RNNs ఉపయోగించి పద స్థాయి టెక్స్ట్ జనరేషన్

[AI for Beginners Curriculum](https://github.com/microsoft/ai-for-beginners) నుండి ల్యాబ్ అసైన్‌మెంట్.

## పని

ఈ ల్యాబ్‌లో, మీరు ఏదైనా పుస్తకాన్ని తీసుకుని, దాన్ని డేటాసెట్‌గా ఉపయోగించి పద స్థాయి టెక్స్ట్ జనరేటర్‌ను ట్రెయిన్ చేయాలి.

## డేటాసెట్

మీకు ఏ పుస్తకాన్ని అయినా ఉపయోగించుకోవచ్చు. మీరు చాలా ఉచిత పాఠ్యాలను [Project Gutenberg](https://www.gutenberg.org/) వద్ద పొందవచ్చు, ఉదాహరణకు, ఇది [Alice's Adventures in Wonderland](https://www.gutenberg.org/files/11/11-0.txt) అనే లూయిస్ క్యారోల్ రచనకు నేరుగా లింక్.
