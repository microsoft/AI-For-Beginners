# మౌంటైన్ కార్ నుండి తప్పించుకోవడానికి శిక్షణ

[AI for Beginners Curriculum](https://github.com/microsoft/ai-for-beginners) నుండి ల్యాబ్ అసైన్‌మెంట్.

## పని

OpenAI పరిసరంలో [Mountain Car](https://www.gymlibrary.ml/environments/classic_control/mountain_car/) ను నియంత్రించడానికి RL ఏజెంట్‌ను శిక్షణ ఇవ్వడం మీ లక్ష్యం.

<img alt="Mountain Car" src="../../../../../../translated_images/te/mountaincar.f7b7a7f6d4f9933b.webp" width="300"/>

## పరిసరము

మౌంటైన్ కార్ పరిసరంలో కారు ఒక లోయలో చిక్కుకున్నది. మీ లక్ష్యం లోయ నుండి దూకి బయటకు వెళ్లి జెండాను చేరుకోవడం. మీరు చేయగల చర్యలు ఎడమకు వేగవంతం చేయడం, కుడికి వేగవంతం చేయడం లేదా ఏమీ చేయకపోవడం. మీరు కార్ యొక్క x-అక్షం పై స్థానం మరియు వేగాన్ని గమనించవచ్చు.

## ప్రారంభ నోట్బుక్

ల్యాబ్ ప్రారంభించడానికి [MountainCar.ipynb](MountainCar.ipynb) ను తెరవండి

## నేర్చుకోవాల్సిన విషయం

ఈ ల్యాబ్ ద్వారా మీరు తెలుసుకోవాలి RL అల్గోరిథమ్స్‌ను కొత్త పరిసరానికి అనుసరించడం సాధారణంగా చాలా సులభమని, ఎందుకంటే OpenAI Gym అన్ని పరిసరాలకు ఒకే ఇంటర్‌ఫేస్ కలిగి ఉంటుంది, మరియు అల్గోరిథమ్స్ ఎక్కువగా పరిసర స్వభావంపై ఆధారపడవు. మీరు Python కోడ్‌ను కూడా ఇలా పునఃరూపకల్పన చేయవచ్చు, ఏ పరిసరాన్ని RL అల్గోరిథమ్‌కు పారామీటర్‌గా పంపగలుగుతారు.
