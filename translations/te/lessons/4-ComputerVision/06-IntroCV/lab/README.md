# ఆప్టికల్ ఫ్లో ఉపయోగించి కదలికలను గుర్తించడం

[AI for Beginners Curriculum](https://aka.ms/ai-beginners) నుండి ల్యాబ్ అసైన్‌మెంట్.

## పని

[ఈ వీడియో](../../../../../../lessons/4-ComputerVision/06-IntroCV/lab/palm-movement.mp4)ను పరిశీలించండి, ఇందులో ఒక వ్యక్తి చేతి తలుపు స్థిరమైన నేపథ్యంపై ఎడమ/కుడి/పై/కింద కదులుతుంది.

<img src="../../../../../../translated_images/te/palm-movement.341495f0e9c47da3.webp" width="30%" alt="Palm Movement Frame"/>

**మీ లక్ష్యం** ఆప్టికల్ ఫ్లో ఉపయోగించి వీడియోలో ఎక్కడ ఎడమ/కుడి/పై/కింద కదలికలు ఉన్నాయో గుర్తించడం.

**విస్తృత లక్ష్యం** చర్మ రంగును ఉపయోగించి చేతి తలుపు/వెళ్ళు కదలికను ట్రాక్ చేయడం, [ఈ బ్లాగ్ పోస్ట్](https://dev.to/amarlearning/finger-detection-and-tracking-using-opencv-and-python-586m) లేదా [ఇక్కడ](http://www.benmeline.com/finger-tracking-with-opencv-and-python/) వివరించినట్లుగా.

## ముఖ్యాంశం

కొన్నిసార్లు, కదలిక గుర్తింపు లేదా వేళ్ల చివరి గుర్తింపు వంటి సాపేక్షంగా క్లిష్టమైన పనులను కంప్యూటర్ విజన్ ద్వారా మాత్రమే పరిష్కరించవచ్చు. అందువల్ల, OpenCV వంటి లైబ్రరీలు ఏమి చేయగలవో తెలుసుకోవడం చాలా ఉపయోగకరం.
