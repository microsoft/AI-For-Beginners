# PyTorch/TensorFlow తో వర్గీకరణ

[AI for Beginners Curriculum](https://github.com/microsoft/ai-for-beginners) నుండి ల్యాబ్ అసైన్‌మెంట్.

## పని

PyTorch లేదా TensorFlow ఉపయోగించి సింగిల్ మరియు మల్టీ-లేయర్డ్ ఫుల్-కనెక్టెడ్ నెట్‌వర్క్‌లను ఉపయోగించి రెండు వర్గీకరణ సమస్యలను పరిష్కరించండి:

1. **[Iris వర్గీకరణ](https://en.wikipedia.org/wiki/Iris_flower_data_set)** సమస్య - ఇది టేబులర్ ఇన్‌పుట్ డేటాతో కూడిన సమస్యకు ఉదాహరణ, ఇది క్లాసికల్ మెషీన్ లెర్నింగ్ ద్వారా నిర్వహించవచ్చు. మీ లక్ష్యం 4 సంఖ్యాత్మక పారామితుల ఆధారంగా ఐరిసులను 3 తరగతులుగా వర్గీకరించడం.
1. మునుపటి సారి చూచిన MNIST చేతితో వ్రాసిన అంకెల వర్గీకరణ సమస్య.

మీకు సాధ్యమైన ఉత్తమ ఖచ్చితత్వాన్ని సాధించడానికి వివిధ నెట్‌వర్క్ నిర్మాణాలను ప్రయత్నించండి.
