# サポート

## 問題の報告とサポートの受け方  

このプロジェクトでは、GitHub Issues を使用してバグや機能リクエストを追跡しています。重複を避けるために、新しい問題を報告する前に既存の問題を検索してください。新しい問題を報告する場合は、バグや機能リクエストを新しい Issue として提出してください。

このプロジェクトの使用に関する質問やサポートについては、ディスカッションボードをご利用ください。
