## [–¢–µ—Å—Ç –ø–µ—Ä–µ–¥ –ª–µ–∫—Ü—ñ—î—é](https://ff-quizzes.netlify.app/en/ai/quiz/11)

–û–¥–Ω–µ –∑ –Ω–∞–π–ø—Ä–æ—Å—Ç—ñ—à–∏—Ö –∑–∞–≤–¥–∞–Ω—å –∫–æ–º–ø'—é—Ç–µ—Ä–Ω–æ–≥–æ –∑–æ—Ä—É ‚Äî —Ü–µ **–∫–ª–∞—Å–∏—Ñ—ñ–∫–∞—Ü—ñ—è –∑–æ–±—Ä–∞–∂–µ–Ω—å**.

–ö–æ–º–ø'—é—Ç–µ—Ä–Ω–∏–π –∑—ñ—Ä —á–∞—Å—Ç–æ –≤–≤–∞–∂–∞—î—Ç—å—Å—è –≥–∞–ª—É–∑–∑—é —à—Ç—É—á–Ω–æ–≥–æ —ñ–Ω—Ç–µ–ª–µ–∫—Ç—É. –°—å–æ–≥–æ–¥–Ω—ñ –±—ñ–ª—å—à—ñ—Å—Ç—å –∑–∞–≤–¥–∞–Ω—å –∫–æ–º–ø'—é—Ç–µ—Ä–Ω–æ–≥–æ –∑–æ—Ä—É –≤–∏—Ä—ñ—à—É—é—Ç—å—Å—è –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –Ω–µ–π—Ä–æ–Ω–Ω–∏—Ö –º–µ—Ä–µ–∂. –ú–∏ –¥—ñ–∑–Ω–∞—î–º–æ—Å—è –±—ñ–ª—å—à–µ –ø—Ä–æ —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–∏–π —Ç–∏–ø –Ω–µ–π—Ä–æ–Ω–Ω–∏—Ö –º–µ—Ä–µ–∂, —è–∫—ñ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è –¥–ª—è –∫–æ–º–ø'—é—Ç–µ—Ä–Ω–æ–≥–æ –∑–æ—Ä—É, [–∑–≥–æ—Ä—Ç–∫–æ–≤—ñ –Ω–µ–π—Ä–æ–Ω–Ω—ñ –º–µ—Ä–µ–∂—ñ](../07-ConvNets/README.md), –ø—Ä–æ—Ç—è–≥–æ–º —Ü—å–æ–≥–æ —Ä–æ–∑–¥—ñ–ª—É.

–û–¥–Ω–∞–∫ –ø–µ—Ä–µ–¥ —Ç–∏–º, —è–∫ –ø–µ—Ä–µ–¥–∞—Ç–∏ –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è –Ω–µ–π—Ä–æ–Ω–Ω—ñ–π –º–µ—Ä–µ–∂—ñ, —É –±–∞–≥–∞—Ç—å–æ—Ö –≤–∏–ø–∞–¥–∫–∞—Ö –º–∞—î —Å–µ–Ω—Å –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–µ—è–∫—ñ –∞–ª–≥–æ—Ä–∏—Ç–º—ñ—á–Ω—ñ –º–µ—Ç–æ–¥–∏ –¥–ª—è –ø–æ–∫—Ä–∞—â–µ–Ω–Ω—è –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è.

–Ü—Å–Ω—É—î –∫—ñ–ª—å–∫–∞ –±—ñ–±–ª—ñ–æ—Ç–µ–∫ Python –¥–ª—è –æ–±—Ä–æ–±–∫–∏ –∑–æ–±—Ä–∞–∂–µ–Ω—å:

* **[imageio](https://imageio.readthedocs.io/en/stable/)** –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è —á–∏—Ç–∞–Ω–Ω—è/–∑–∞–ø–∏—Å—É —Ä—ñ–∑–Ω–∏—Ö —Ñ–æ—Ä–º–∞—Ç—ñ–≤ –∑–æ–±—Ä–∞–∂–µ–Ω—å. –í–æ–Ω–∞ —Ç–∞–∫–æ–∂ –ø—ñ–¥—Ç—Ä–∏–º—É—î ffmpeg ‚Äî –∫–æ—Ä–∏—Å–Ω–∏–π —ñ–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç –¥–ª—è –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –∫–∞–¥—Ä—ñ–≤ –≤—ñ–¥–µ–æ –≤ –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è.
* **[Pillow](https://pillow.readthedocs.io/en/stable/index.html)** (—Ç–∞–∫–æ–∂ –≤—ñ–¥–æ–º–∞ —è–∫ PIL) —î —Ç—Ä–æ—Ö–∏ –ø–æ—Ç—É–∂–Ω—ñ—à–æ—é —ñ –ø—ñ–¥—Ç—Ä–∏–º—É—î –¥–µ—è–∫—ñ –º–∞–Ω—ñ–ø—É–ª—è—Ü—ñ—ó —ñ–∑ –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è–º–∏, —Ç–∞–∫—ñ —è–∫ –º–æ—Ä—Ñ—ñ–Ω–≥, –Ω–∞–ª–∞—à—Ç—É–≤–∞–Ω–Ω—è –ø–∞–ª—ñ—Ç—Ä–∏ —Ç–æ—â–æ.
* **[OpenCV](https://opencv.org/)** ‚Äî —Ü–µ –ø–æ—Ç—É–∂–Ω–∞ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ –æ–±—Ä–æ–±–∫–∏ –∑–æ–±—Ä–∞–∂–µ–Ω—å, –Ω–∞–ø–∏—Å–∞–Ω–∞ –Ω–∞ C++, —è–∫–∞ —Å—Ç–∞–ª–∞ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º *–¥–µ-—Ñ–∞–∫—Ç–æ* –¥–ª—è –æ–±—Ä–æ–±–∫–∏ –∑–æ–±—Ä–∞–∂–µ–Ω—å. –í–æ–Ω–∞ –º–∞—î –∑—Ä—É—á–Ω–∏–π —ñ–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è Python.
* **[dlib](http://dlib.net/)** ‚Äî —Ü–µ –±—ñ–±–ª—ñ–æ—Ç–µ–∫–∞ C++, —è–∫–∞ —Ä–µ–∞–ª—ñ–∑—É—î –±–∞–≥–∞—Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º—ñ–≤ –º–∞—à–∏–Ω–Ω–æ–≥–æ –Ω–∞–≤—á–∞–Ω–Ω—è, –≤–∫–ª—é—á–∞—é—á–∏ –¥–µ—è–∫—ñ –∞–ª–≥–æ—Ä–∏—Ç–º–∏ –∫–æ–º–ø'—é—Ç–µ—Ä–Ω–æ–≥–æ –∑–æ—Ä—É. –í–æ–Ω–∞ —Ç–∞–∫–æ–∂ –º–∞—î —ñ–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è Python —ñ –º–æ–∂–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏—Å—è –¥–ª—è —Å–∫–ª–∞–¥–Ω–∏—Ö –∑–∞–≤–¥–∞–Ω—å, —Ç–∞–∫–∏—Ö —è–∫ –≤–∏—è–≤–ª–µ–Ω–Ω—è –æ–±–ª–∏—á—á—è —Ç–∞ –≤–∏–∑–Ω–∞—á–µ–Ω–Ω—è –∫–ª—é—á–æ–≤–∏—Ö —Ç–æ—á–æ–∫ –æ–±–ª–∏—á—á—è.

## OpenCV

[OpenCV](https://opencv.org/) –≤–≤–∞–∂–∞—î—Ç—å—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º *–¥–µ-—Ñ–∞–∫—Ç–æ* –¥–ª—è –æ–±—Ä–æ–±–∫–∏ –∑–æ–±—Ä–∞–∂–µ–Ω—å. –í–æ–Ω–∞ –º—ñ—Å—Ç–∏—Ç—å –±–∞–≥–∞—Ç–æ –∫–æ—Ä–∏—Å–Ω–∏—Ö –∞–ª–≥–æ—Ä–∏—Ç–º—ñ–≤, —Ä–µ–∞–ª—ñ–∑–æ–≤–∞–Ω–∏—Ö –Ω–∞ C++. –í–∏ —Ç–∞–∫–æ–∂ –º–æ–∂–µ—Ç–µ –≤–∏–∫–ª–∏–∫–∞—Ç–∏ OpenCV –∑ Python.

–•–æ—Ä–æ—à–∏–º –º—ñ—Å—Ü–µ–º –¥–ª—è –≤–∏–≤—á–µ–Ω–Ω—è OpenCV —î [–∫—É—Ä—Å Learn OpenCV](https://learnopencv.com/getting-started-with-opencv/). –£ –Ω–∞—à—ñ–π –ø—Ä–æ–≥—Ä–∞–º—ñ –º–∏ –Ω–µ —Å—Ç–∞–≤–∏–º–æ –∑–∞ –º–µ—Ç—É –≤–∏–≤—á–∏—Ç–∏ OpenCV, –∞ –ª–∏—à–µ –ø–æ–∫–∞–∑–∞—Ç–∏ –≤–∞–º –∫—ñ–ª—å–∫–∞ –ø—Ä–∏–∫–ª–∞–¥—ñ–≤, –∫–æ–ª–∏ —ó—ó –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —ñ —è–∫.

### –ó–∞–≤–∞–Ω—Ç–∞–∂–µ–Ω–Ω—è –∑–æ–±—Ä–∞–∂–µ–Ω—å

–ó–æ–±—Ä–∞–∂–µ–Ω–Ω—è –≤ Python –∑—Ä—É—á–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—Ç–∏ —É –≤–∏–≥–ª—è–¥—ñ –º–∞—Å–∏–≤—ñ–≤ NumPy. –ù–∞–ø—Ä–∏–∫–ª–∞–¥, –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è –≤ –≥—Ä–∞–¥–∞—Ü—ñ—è—Ö —Å—ñ—Ä–æ–≥–æ —Ä–æ–∑–º—ñ—Ä–æ–º 320x200 –ø—ñ–∫—Å–µ–ª—ñ–≤ –±—É–¥–µ –∑–±–µ—Ä—ñ–≥–∞—Ç–∏—Å—è —É –≤–∏–≥–ª—è–¥—ñ –º–∞—Å–∏–≤—É 200x320, –∞ –∫–æ–ª—å–æ—Ä–æ–≤—ñ –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è —Ç–æ–≥–æ –∂ —Ä–æ–∑–º—ñ—Ä—É –º–∞—Ç–∏–º—É—Ç—å —Ñ–æ—Ä–º—É 200x320x3 (–¥–ª—è 3 –∫–æ–ª—å–æ—Ä–æ–≤–∏—Ö –∫–∞–Ω–∞–ª—ñ–≤). –©–æ–± –∑–∞–≤–∞–Ω—Ç–∞–∂–∏—Ç–∏ –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è, –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏ –Ω–∞—Å—Ç—É–ø–Ω–∏–π –∫–æ–¥:

```python
import cv2

im = cv2.imread('image.jpeg')
plt.imshow(im)
```

–¢—Ä–∞–¥–∏—Ü—ñ–π–Ω–æ OpenCV –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î –∫–æ–¥—É–≤–∞–Ω–Ω—è BGR (—Å–∏–Ω—ñ–π-–∑–µ–ª–µ–Ω–∏–π-—á–µ—Ä–≤–æ–Ω–∏–π) –¥–ª—è –∫–æ–ª—å–æ—Ä–æ–≤–∏—Ö –∑–æ–±—Ä–∞–∂–µ–Ω—å, —Ç–æ–¥—ñ —è–∫ —Ä–µ—à—Ç–∞ —ñ–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ñ–≤ Python –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å –±—ñ–ª—å—à —Ç—Ä–∞–¥–∏—Ü—ñ–π–Ω–µ RGB (—á–µ—Ä–≤–æ–Ω–∏–π-–∑–µ–ª–µ–Ω–∏–π-—Å–∏–Ω—ñ–π). –©–æ–± –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è –≤–∏–≥–ª—è–¥–∞–ª–æ –ø—Ä–∞–≤–∏–ª—å–Ω–æ, –ø–æ—Ç—Ä—ñ–±–Ω–æ –ø–µ—Ä–µ—Ç–≤–æ—Ä–∏—Ç–∏ –π–æ–≥–æ –≤ –∫–æ–ª—ñ—Ä–Ω–∏–π –ø—Ä–æ—Å—Ç—ñ—Ä RGB, –∞–±–æ —à–ª—è—Ö–æ–º –ø–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫–∏ –≤–∏–º—ñ—Ä—ñ–≤ —É –º–∞—Å–∏–≤—ñ NumPy, –∞–±–æ –≤–∏–∫–ª–∏–∫–æ–º —Ñ—É–Ω–∫—Ü—ñ—ó OpenCV:

```python

```

–¢—É –∂ —Ñ—É–Ω–∫—Ü—ñ—é `cvtColor` –º–æ–∂–Ω–∞ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –¥–ª—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è —ñ–Ω—à–∏—Ö –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω—å –∫–æ–ª—ñ—Ä–Ω–æ–≥–æ –ø—Ä–æ—Å—Ç–æ—Ä—É, —Ç–∞–∫–∏—Ö —è–∫ –ø–µ—Ä–µ—Ç–≤–æ—Ä–µ–Ω–Ω—è –∑–æ–±—Ä–∞–∂–µ–Ω–Ω—è –≤ –≥—Ä–∞–¥–∞—Ü—ñ—ó —Å—ñ—Ä–æ–≥–æ –∞–±–æ –≤ HSV (–≤—ñ–¥—Ç—ñ–Ω–æ–∫-–Ω–∞—Å–∏—á–µ–Ω—ñ—Å—Ç—å-–∑–Ω–∞—á–µ–Ω–Ω—è).

–í–∏ —Ç–∞–∫–æ–∂ –º–æ–∂–µ—Ç–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ OpenCV –¥–ª—è –∑–∞–≤–∞–Ω—Ç–∞–∂–µ–Ω–Ω—è –≤—ñ–¥–µ–æ –∫–∞–¥—Ä –∑–∞ –∫–∞–¥—Ä–æ–º ‚Äî –ø—Ä–∏–∫–ª–∞–¥ –Ω–∞–≤–µ–¥–µ–Ω–æ —É –≤–ø—Ä–∞–≤—ñ [OpenCV Notebook](OpenCV.ipynb).
