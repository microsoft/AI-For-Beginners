# 支持

## 如何提交问题和获取帮助  

此项目使用 GitHub Issues 来跟踪错误和功能请求。在提交新问题之前，请先搜索现有问题以避免重复。对于新问题，请将您的错误或功能请求作为新问题提交。

如果需要帮助或对使用此项目有疑问，请使用讨论板。
