# 作业：笔记本

使用与本课程相关的笔记本（可以是 PyTorch 或 TensorFlow 版本），使用您自己的数据集重新运行它们，例如来自 Kaggle 的数据集，并注明出处。重写笔记本以突出您自己的发现。尝试使用不同类型的数据集并记录您的发现，例如使用[这些 Beatles 歌词](https://www.kaggle.com/datasets/jenlooper/beatles-lyrics)中的文本。
