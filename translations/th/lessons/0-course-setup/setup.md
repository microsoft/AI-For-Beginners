# เริ่มต้นกับหลักสูตรนี้

## คุณเป็นนักเรียนหรือเปล่า?

เริ่มต้นด้วยแหล่งข้อมูลต่อไปนี้:

* [หน้าศูนย์รวมสำหรับนักเรียน](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) ในหน้านี้ คุณจะพบกับแหล่งข้อมูลสำหรับผู้เริ่มต้น ชุดเครื่องมือสำหรับนักเรียน และแม้กระทั่งวิธีการรับบัตรกำนัลสอบฟรี นี่คือหน้าที่คุณควรบุ๊กมาร์กไว้และกลับมาตรวจสอบเป็นระยะๆ เพราะเรามีการอัปเดตเนื้อหาอย่างน้อยทุกเดือน
* [Microsoft Student Learn Ambassadors](https://studentambassadors.microsoft.com?WT.mc_id=academic-77998-cacaste) เข้าร่วมชุมชนระดับโลกของนักเรียนที่เป็นตัวแทน นี่อาจเป็นทางเข้าสู่ Microsoft ของคุณ

**นักเรียน** มีสองสามวิธีที่คุณสามารถใช้หลักสูตรนี้ได้ อย่างแรก คุณสามารถอ่านเนื้อหาและดูโค้ดได้โดยตรงบน GitHub หากคุณต้องการรันโค้ดในโน้ตบุ๊กใดๆ - [อ่านคำแนะนำของเรา](./how-to-run.md) และหาคำแนะนำเพิ่มเติมเกี่ยวกับวิธีการทำได้ [ในบล็อกโพสต์นี้](https://soshnikov.com/education/how-to-execute-notebooks-from-github/)

> **Note**: [คำแนะนำเกี่ยวกับวิธีการรันโค้ดในหลักสูตรนี้](./how-to-run.md)

## การเรียนด้วยตัวเอง

อย่างไรก็ตาม หากคุณต้องการเรียนหลักสูตรนี้เป็นโครงการเรียนด้วยตัวเอง เราแนะนำให้คุณ fork repo ทั้งหมดไปยังบัญชี GitHub ของคุณเองและทำแบบฝึกหัดด้วยตัวเองหรือกับกลุ่ม:

* เริ่มต้นด้วยแบบทดสอบก่อนการบรรยาย
* อ่านข้อความแนะนำสำหรับการบรรยาย
* หากการบรรยายมีโน้ตบุ๊กเพิ่มเติม ให้ผ่านไปทีละส่วน อ่านและรันโค้ด หากมีโน้ตบุ๊กทั้ง TensorFlow และ PyTorch ให้คุณเลือกโฟกัสที่อันใดอันหนึ่ง - เลือกเฟรมเวิร์กที่คุณชอบ
* โน้ตบุ๊กมักจะมีความท้าทายบางอย่างที่ต้องการให้คุณปรับแต่งโค้ดเล็กน้อยเพื่อทดลอง
* ทำแบบทดสอบหลังการบรรยาย
* หากมีห้องปฏิบัติการที่แนบมากับโมดูล - ทำการบ้านให้เสร็จ
* เยี่ยมชม [กระดานสนทนา](https://github.com/microsoft/AI-For-Beginners/discussions) เพื่อ "เรียนรู้แบบเปิดเผย"

> สำหรับการศึกษาต่อ เราแนะนำให้ติดตาม [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) โมดูลและเส้นทางการเรียนรู้เหล่านี้

**ครู** เราได้ [รวมคำแนะนำบางส่วน](./for-teachers.md) เกี่ยวกับวิธีการใช้หลักสูตรนี้

## วิธีการสอน

เราได้เลือกหลักการสอนสองข้อในขณะที่สร้างหลักสูตรนี้: การทำให้เป็น **โครงการที่เน้นการปฏิบัติ** และการรวม **แบบทดสอบบ่อยครั้ง**

โดยการทำให้เนื้อหาสอดคล้องกับโครงการ กระบวนการจะน่าสนใจยิ่งขึ้นสำหรับนักเรียนและช่วยเพิ่มการจดจำแนวคิด นอกจากนี้ แบบทดสอบที่มีความเสี่ยงต่ำก่อนชั้นเรียนจะช่วยตั้งเป้าหมายให้นักเรียนเรียนรู้หัวข้อหนึ่งๆ ในขณะที่แบบทดสอบที่สองหลังชั้นเรียนจะช่วยเพิ่มการจดจำเพิ่มเติม หลักสูตรนี้ถูกออกแบบมาให้ยืดหยุ่นและสนุกสนาน และสามารถเรียนได้ทั้งหมดหรือบางส่วน โครงการเริ่มต้นจากขนาดเล็กและซับซ้อนขึ้นเรื่อยๆ จนถึงสิ้นสุดรอบ 12 สัปดาห์

> **หมายเหตุเกี่ยวกับแบบทดสอบ**: แบบทดสอบทั้งหมดอยู่ใน [แอปนี้](https://red-field-0a6ddfd03.1.azurestaticapps.net/) มีทั้งหมด 50 แบบทดสอบ แต่ละแบบมี 3 คำถาม แบบทดสอบเหล่านี้ถูกลิงก์จากในบทเรียน แต่แอปแบบทดสอบสามารถรันได้ในเครื่องของคุณเอง; ทำตามคำแนะนำในโฟลเดอร์ `etc/quiz-app`

## การเข้าถึงแบบออฟไลน์

คุณสามารถรันเอกสารนี้แบบออฟไลน์ได้โดยใช้ [Docsify](https://docsify.js.org/#/) Fork repo นี้, [ติดตั้ง Docsify](https://docsify.js.org/#/quickstart) บนเครื่องของคุณ และจากนั้นในโฟลเดอร์รากของ repo นี้ ให้พิมพ์ `docsify serve` เว็บไซต์จะถูกให้บริการบนพอร์ต 3000 บน localhost ของคุณ: `localhost:3000` ไฟล์ pdf ของหลักสูตรสามารถดาวน์โหลดได้ [ที่ลิงก์นี้](../../../../../../../../../etc/pdf/readme.pdf)
