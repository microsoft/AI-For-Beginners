# ဒီသင်ရိုးကို စတင်အသုံးပြုခြင်း

## သင်ကျောင်းသားလား?

အောက်ပါအရင်းအမြစ်များနှင့် စတင်ပါ:

* [ကျောင်းသား Hub စာမျက်နှာ](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) ဒီစာမျက်နှာမှာ သင်တန်းသားများအတွက် အခြေခံအရင်းအမြစ်များ၊ ကျောင်းသားအထုပ်များ၊ အခမဲ့လက်မှတ်ရယူနည်းများကို တွေ့နိုင်ပါတယ်။ ဒီစာမျက်နှာကို Bookmark လုပ်ထားပြီး အချိန်အခါမရွေး ပြန်လည်ကြည့်ရှုပါ၊ အကြောင်းမူတည်ပြီး အကြောင်းအရာများကို လစဉ်ပြောင်းလဲတင်ဆက်လေ့ရှိပါတယ်။
* [Microsoft ကျောင်းသားသင်ကြားမှု သံတမန်များ](https://studentambassadors.microsoft.com?WT.mc_id=academic-77998-cacaste) ကျောင်းသားသံတမန်များ၏ ကမ္ဘာလုံးဆိုင်ရာအသိုင်းအဝိုင်းကို ဝင်ရောက်ပါ၊ ဒါဟာ Microsoft ကို ဝင်ရောက်ဖို့ သင့်အတွက် အခွင့်အလမ်းတစ်ခုဖြစ်နိုင်ပါတယ်။

**ကျောင်းသားများ**၊ ဒီသင်ရိုးကို အသုံးပြုဖို့ နည်းလမ်းအနည်းငယ်ရှိပါတယ်။ ပထမဦးစွာ GitHub ပေါ်မှာ တိုက်ရိုက်စာသားများကို ဖတ်ပြီး ကုဒ်များကို ကြည့်ရှုနိုင်ပါတယ်။ သင် notebooks များထဲက ကုဒ်များကို run လုပ်ချင်တယ်ဆိုရင် - [ကျွန်ုပ်တို့၏ညွှန်ကြားချက်များကို ဖတ်ပါ](./how-to-run.md)၊ နောက်ထပ်အကြံဉာဏ်များကို [ဒီဘလော့ဂ်ပို့စ်](https://soshnikov.com/education/how-to-execute-notebooks-from-github/) မှာ ရှာဖွေပါ။

> **Note**: [ဒီသင်ရိုးထဲက ကုဒ်ကို run လုပ်နည်းအကြောင်း ညွှန်ကြားချက်များ](./how-to-run.md)

## ကိုယ်တိုင်လေ့လာခြင်း

သို့သော် သင်ဒီသင်တန်းကို ကိုယ်တိုင်လေ့လာမှု project အဖြစ် လုပ်ချင်တယ်ဆိုရင်၊ အပြည့်အစုံ repo ကို သင့် GitHub အကောင့်သို့ fork လုပ်ပြီး ကိုယ်တိုင် (သို့မဟုတ်) အဖွဲ့ဖြင့် လေ့လာနိုင်ပါတယ်:

* သင်ခန်းစာမတိုင်မီ စစ်တမ်းကို စတင်ပါ။
* သင်ခန်းစာအတွက် အကျဉ်းစာသားကို ဖတ်ပါ။
* သင်ခန်းစာမှာ ထပ်ဆောင်း notebooks ရှိရင်၊ အဲဒီအတိုင်း ဖတ်ပြီး ကုဒ်ကို run လုပ်ပါ။ TensorFlow နဲ့ PyTorch notebooks နှစ်ခုလုံးပေးထားရင်၊ သင့်အကြိုက်ဆုံး framework ကို ရွေးပြီး အာရုံစိုက်နိုင်ပါတယ်။
* Notebooks တွင် အခက်အခဲအချို့ပါဝင်ပြီး၊ ကုဒ်ကို အနည်းငယ်ပြောင်းလဲပြီး စမ်းသပ်ဖို့ လိုအပ်နိုင်ပါတယ်။
* သင်ခန်းစာပြီးနောက် စစ်တမ်းကို ဖြေပါ။
* Module နဲ့ ဆက်စပ် lab ရှိရင် - အလုပ်ကို ပြီးစီးပါ။
* [ဆွေးနွေးမှုဘုတ်](https://github.com/microsoft/AI-For-Beginners/discussions) ကို သွားရောက်ပြီး "အသံထွက်လေ့လာမှု" ပြုလုပ်ပါ။

> နောက်ထပ်လေ့လာရန်အတွက် [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) modules နဲ့ learning paths များကို လိုက်နာရန် အကြံပြုပါသည်။

**ဆရာများ**၊ ဒီသင်ရိုးကို ဘယ်လိုအသုံးပြုရမလဲဆိုတာအတွက် [အကြံပြုချက်အချို့](./for-teachers.md) ထည့်သွင်းထားပါတယ်။

## သင်ကြားမှုနည်းစနစ်

ဒီသင်ရိုးကို တည်ဆောက်ရာမှာ သင်ကြားမှုနည်းစနစ်နှစ်ခုကို ရွေးချယ်ထားပါတယ် - **project-based** ဖြစ်စေခြင်းနှင့် **မကြာခဏ စစ်တမ်းများ** ပါဝင်စေခြင်း။

အကြောင်းအရာများကို project များနှင့် ကိုက်ညီစေခြင်းအားဖြင့် ကျောင်းသားများအတွက် စိတ်ဝင်စားမှုကို တိုးမြှင့်ပြီး အကြောင်းအရာများကို မှတ်မိမှုကို တိုးတက်စေပါသည်။ ထို့အပြင်၊ သင်ခန်းစာမတိုင်မီ စစ်တမ်းတစ်ခုက ကျောင်းသား၏ အာရုံစိုက်မှုကို သင်ခန်းစာအပေါ် ဦးတည်စေပြီး၊ သင်ခန်းစာပြီးနောက် စစ်တမ်းတစ်ခုက အကြောင်းအရာများကို နောက်ထပ် မှတ်မိစေပါသည်။ ဒီသင်ရိုးကို အပြည့်အစုံ သို့မဟုတ် အစိတ်အပိုင်းအချို့အဖြစ် flexible နဲ့ ပျော်ရွှင်စွာ လေ့လာနိုင်အောင် ဒီဇိုင်းထုတ်ထားပါတယ်။ Project များကို သေးငယ်စွာစတင်ပြီး 12 ပတ်အတွင်း အဆင့်မြင့်ဖြစ်လာစေပါသည်။

> **စစ်တမ်းများအကြောင်း မှတ်ချက်**: စစ်တမ်းအားလုံးကို [ဒီ app](https://red-field-0a6ddfd03.1.azurestaticapps.net/) ထဲမှာ ထည့်သွင်းထားပြီး၊ စုစုပေါင်း 50 စစ်တမ်း၊ တစ်ခုလျှင် မေးခွန်း 3 ခုပါဝင်ပါတယ်။ Lessons ထဲကနေ link လုပ်ထားပြီး၊ quiz app ကို locally run လုပ်နိုင်ပါတယ်။ `etc/quiz-app` folder ထဲက ညွှန်ကြားချက်များကို လိုက်နာပါ။

## အော့ဖ်လိုင်း အသုံးပြုမှု

ဒီစာရွက်စာတမ်းကို [Docsify](https://docsify.js.org/#/) အသုံးပြုပြီး အော့ဖ်လိုင်းမှာ run လုပ်နိုင်ပါတယ်။ ဒီ repo ကို fork လုပ်ပြီး၊ [Docsify ကို install လုပ်ပါ](https://docsify.js.org/#/quickstart) သင့် local machine ပေါ်မှာ၊ ပြီးရင် ဒီ repo ရဲ့ root folder မှာ `docsify serve` ကို ရိုက်ထည့်ပါ။ Website ကို သင့် localhost ရဲ့ port 3000 မှာ serve လုပ်ပါမည်: `localhost:3000`။ သင်ရိုးရဲ့ pdf ကို [ဒီ link](../../../../../../../../../etc/pdf/readme.pdf) မှာ ရနိုင်ပါတယ်။
