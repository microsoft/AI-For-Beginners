**നിങ്ങളുടെ ലക്ഷ്യം** ഓപ്റ്റിക്കൽ ഫ്ലോ ഉപയോഗിച്ച് വീഡിയോയിലെ ഏത് ഭാഗങ്ങളിൽ മുകളിലേക്ക്/താഴേക്ക്/ഇടത്തേക്ക്/വലത്തേക്ക് ചലനങ്ങൾ ഉണ്ടെന്ന് കണ്ടെത്താൻ കഴിയുക എന്നതാണ്.

**വിപുലീകരണ ലക്ഷ്യം** ചർമ്മത്തിന്റെ നിറം ഉപയോഗിച്ച് കൈവിരൽ/വിരൽ ചലനം ട്രാക്ക് ചെയ്യുക, [ഈ ബ്ലോഗ് പോസ്റ്റിൽ](https://dev.to/amarlearning/finger-detection-and-tracking-using-opencv-and-python-586m) അല്ലെങ്കിൽ [ഇവിടെ](http://www.benmeline.com/finger-tracking-with-opencv-and-python/) വിവരിച്ചിരിക്കുന്നതുപോലെ.
