## ടാസ്‌ക്

ഒരു പെട്ട് നഴ്സറിയുടെ എല്ലാ പെട്ടുകളെയും കാറ്റലോഗ് ചെയ്യാനുള്ള ഒരു ആപ്ലിക്കേഷൻ വികസിപ്പിക്കേണ്ടതായി നിങ്ങൾക്ക് കരുതുക. അത്തരത്തിലുള്ള ഒരു ആപ്ലിക്കേഷന്റെ മികച്ച സവിശേഷതകളിൽ ഒന്ന് ഫോട്ടോഗ്രാഫിൽ നിന്ന് സ്വയം ജാതി കണ്ടെത്തുക എന്നതാണ്. ഈ അസൈൻമെന്റിൽ, നാം ട്രാൻസ്ഫർ ലേണിംഗ് ഉപയോഗിച്ച് [Oxford-IIIT](https://www.robots.ox.ac.uk/~vgg/data/pets/) പെട്ട് ഡാറ്റാസെറ്റിൽ നിന്നുള്ള യഥാർത്ഥ ജീവിത പെട്ട് ചിത്രങ്ങളെ വർഗ്ഗീകരിക്കും.
