# RNN ഉപയോഗിച്ച് വാക്ക് നിലയിലെ ടെക്സ്റ്റ് ജനറേഷൻ

[AI for Beginners Curriculum](https://github.com/microsoft/ai-for-beginners) ലെ ലാബ് അസൈൻമെന്റ്.

## ടാസ്‌ക്

ഈ ലാബിൽ, നിങ്ങൾക്ക് ഏതെങ്കിലും ഒരു പുസ്തകം എടുത്ത് അതിനെ ഡാറ്റാസെറ്റായി ഉപയോഗിച്ച് വാക്ക് നിലയിലെ ടെക്സ്റ്റ് ജനറേറ്റർ ട്രെയിൻ ചെയ്യണം.

## ഡാറ്റാസെറ്റ്

നിങ്ങൾക്ക് ഏതെങ്കിലും ഒരു പുസ്തകം ഉപയോഗിക്കാം. [Project Gutenberg](https://www.gutenberg.org/) ൽ നിങ്ങൾക്ക് നിരവധി സൗജന്യ ഗ്രന്ഥങ്ങൾ ലഭ്യമാണ്, ഉദാഹരണത്തിന്, ലൂയിസ് കാരോൾ എഴുതിയ [Alice's Adventures in Wonderland](https://www.gutenberg.org/files/11/11-0.txt) എന്ന ഗ്രന്ഥത്തിലേക്ക് നേരിട്ട് ലിങ്ക് ഇതാ.
