# അസൈൻമെന്റ്: ട്രാൻസ്ഫോർമേഴ്സ്

HuggingFace-ൽ ട്രാൻസ്ഫോർമേഴ്സുമായി പരീക്ഷണം നടത്തൂ! അവരുടെ സൈറ്റിൽ ലഭ്യമായ വിവിധ മോഡലുകളുമായി പ്രവർത്തിക്കാൻ അവർ നൽകുന്ന ചില സ്ക്രിപ്റ്റുകൾ പരീക്ഷിക്കൂ: https://huggingface.co/docs/transformers/run_scripts. അവരുടെ ഡാറ്റാസെറ്റുകളിൽ ഒന്നിനെ പരീക്ഷിച്ച്, പിന്നീട് ഈ പാഠ്യപദ്ധതിയിൽ നിന്നോ Kaggle-ൽ നിന്നോ നിങ്ങളുടെ സ്വന്തം ഡാറ്റാസെറ്റ് ഇറക്കുമതി ചെയ്ത്, രസകരമായ വാചകങ്ങൾ സൃഷ്ടിക്കാൻ കഴിയുന്നുണ്ടോ എന്ന് നോക്കൂ. നിങ്ങളുടെ കണ്ടെത്തലുകളുമായി ഒരു നോട്ട്‌ബുക്ക് തയ്യാറാക്കൂ.
