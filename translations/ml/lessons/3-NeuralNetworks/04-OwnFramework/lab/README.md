## ചോദ്യങ്ങൾ

ഈ ലാബിന്റെ ഫലമായി, താഴെപ്പറയുന്ന ചോദ്യങ്ങൾക്ക് ഉത്തരം നൽകാൻ ശ്രമിക്കുക:

- ഇന്റർ-ലെയർ ആക്ടിവേഷൻ ഫംഗ്ഷൻ നെറ്റ്‌വർക്ക് പ്രകടനത്തെ ബാധിക്കുന്നുണ്ടോ?
- ഈ ടാസ്‌കിനായി 2-അഥവാ 3-ലെയർ നെറ്റ്‌വർക്ക് ആവശ്യമുണ്ടോ?
- നെറ്റ്‌വർക്ക് പരിശീലിപ്പിക്കുമ്പോൾ നിങ്ങൾക്ക് എന്തെങ്കിലും പ്രശ്നങ്ങൾ അനുഭവപ്പെട്ടോ? പ്രത്യേകിച്ച് ലെയറുകളുടെ എണ്ണം വർദ്ധിച്ചപ്പോൾ.
- പരിശീലനത്തിനിടെ നെറ്റ്‌വർക്ക് വെയ്റ്റുകൾ എങ്ങനെ പെരുമാറുന്നു? ബന്ധം മനസിലാക്കാൻ വെയ്റ്റുകളുടെ പരമാവധി ആബ്സല്യൂട്ട് മൂല്യം എപ്പോക്കിനെതിരെ പ്ലോട്ട് ചെയ്യാവുന്നതാണ്.
