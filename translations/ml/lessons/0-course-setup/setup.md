# ഈ പാഠ്യപദ്ധതിയുമായി ആരംഭിക്കുക

## നിങ്ങൾ ഒരു വിദ്യാർത്ഥിയാണോ?

താഴെപ്പറയുന്ന വിഭവങ്ങൾ ഉപയോഗിച്ച് ആരംഭിക്കുക:

* [Student Hub പേജ്](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) ഈ പേജിൽ, നിങ്ങൾക്ക് തുടക്കക്കാരുടെ വിഭവങ്ങൾ, Student പാക്കുകൾ, കൂടാതെ സൗജന്യ സർട്ടിഫിക്കറ്റ് വൗച്ചർ നേടാനുള്ള മാർഗങ്ങളും ലഭിക്കും. മാസത്തിൽ കുറഞ്ഞത് ഒരിക്കൽ ഉള്ളടക്കം മാറ്റുന്നതുകൊണ്ട് ഈ പേജ് ബുക്ക്മാർക്ക് ചെയ്ത് ഇടയ്ക്കിടെ പരിശോധിക്കേണ്ടതാണ്.
* [Microsoft Student Learn അംബാസഡർമാർ](https://studentambassadors.microsoft.com?WT.mc_id=academic-77998-cacaste) ഒരു ആഗോള വിദ്യാർത്ഥി അംബാസഡർ സമൂഹത്തിൽ ചേരുക, ഇത് Microsoft-ലേക്ക് നിങ്ങളുടെ വഴി ആകാം.

**വിദ്യാർത്ഥികൾ**, പാഠ്യപദ്ധതി ഉപയോഗിക്കുന്നതിന് ചില മാർഗങ്ങളുണ്ട്. ആദ്യം, നിങ്ങൾക്ക് GitHub-ൽ നേരിട്ട് ടെക്സ്റ്റ് വായിച്ച് കോഡ് പരിശോധിക്കാം. നോട്ട്‌ബുക്കുകളിൽ കോഡ് പ്രവർത്തിപ്പിക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ - [നമ്മുടെ നിർദ്ദേശങ്ങൾ വായിക്കുക](./etc/how-to-run.md), കൂടാതെ ഇത് എങ്ങനെ ചെയ്യാമെന്ന് [ഈ ബ്ലോഗ് പോസ്റ്റിൽ](https://soshnikov.com/education/how-to-execute-notebooks-from-github/) കൂടുതൽ ഉപദേശം കണ്ടെത്തുക.

> **കുറിപ്പ്**: [ഈ പാഠ്യപദ്ധതിയിലെ കോഡ് എങ്ങനെ പ്രവർത്തിപ്പിക്കാമെന്ന് നിർദ്ദേശങ്ങൾ](./how-to-run.md)

## സ്വയം പഠനം

എങ്കിൽ, നിങ്ങൾ കോഴ്സ് സ്വയം പഠന പദ്ധതിയായി എടുക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, ഈ റീപ്പോ നിങ്ങളുടെ സ്വന്തം GitHub അക്കൗണ്ടിലേക്ക് ഫോർക്ക് ചെയ്ത്, വ്യായാമങ്ങൾ സ്വയം അല്ലെങ്കിൽ ഒരു ഗ്രൂപ്പിനൊപ്പം പൂർത്തിയാക്കാൻ ഞങ്ങൾ ശുപാർശ ചെയ്യുന്നു:

* പ്രീ-ലെക്ചർ ക്വിസ് ആരംഭിക്കുക.
* ലെക്ചറിന്റെ പരിചയപ്പെടുത്തൽ ടെക്സ്റ്റ് വായിക്കുക.
* ലെക്ചറിന് അധിക നോട്ട്‌ബുക്കുകൾ ഉണ്ടെങ്കിൽ, അവ പരിശോധിച്ച് കോഡ് വായിക്കുകയും പ്രവർത്തിപ്പിക്കുകയും ചെയ്യുക. TensorFlow-യും PyTorch-ഉം ഉള്ള നോട്ട്‌ബുക്കുകൾ നൽകിയിട്ടുണ്ടെങ്കിൽ, നിങ്ങൾക്ക് ഇഷ്ടപ്പെട്ട ഫ്രെയിംവർക്ക് തിരഞ്ഞെടുക്കാം.
* നോട്ട്‌ബുക്കുകളിൽ ചില പരീക്ഷണങ്ങൾ ഉണ്ടാകാം, അവയിൽ കോഡ് ചെറിയ മാറ്റങ്ങൾ വരുത്തി പരീക്ഷിക്കേണ്ടതുണ്ട്.
* പോസ്റ്റ്-ലെക്ചർ ക്വിസ് എടുക്കുക.
* മോഡ്യൂളിനൊപ്പം ലാബ് ഉണ്ടെങ്കിൽ - അസൈൻമെന്റ് പൂർത്തിയാക്കുക.
* "Learn out loud" ചെയ്യാൻ [Discussion board](https://github.com/microsoft/AI-For-Beginners/discussions) സന്ദർശിക്കുക.

> കൂടുതൽ പഠനത്തിന്, ഈ [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) മോഡ്യൂളുകളും പഠന പാതകളും പിന്തുടരാൻ ഞങ്ങൾ ശുപാർശ ചെയ്യുന്നു.

**അധ്യാപകർ**, ഈ പാഠ്യപദ്ധതി ഉപയോഗിക്കുന്നതിന് ഞങ്ങൾ [ചില നിർദ്ദേശങ്ങൾ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്](/for-teachers.md).

## പാഠശാസ്ത്രം

ഈ പാഠ്യപദ്ധതി നിർമ്മിക്കുമ്പോൾ ഞങ്ങൾ രണ്ട് പാഠശാസ്ത്ര തത്വങ്ങൾ തിരഞ്ഞെടുക്കുകയുണ്ടായി: പ്രായോഗികമായ **പ്രോജക്ട് അടിസ്ഥാനമാക്കിയുള്ള** പഠനം ഉറപ്പാക്കുക, കൂടാതെ **സാധാരണ ക്വിസുകൾ** ഉൾപ്പെടുത്തുക.

ഉള്ളടക്കം പ്രോജക്ടുകളുമായി പൊരുത്തപ്പെടുന്നതിലൂടെ, വിദ്യാർത്ഥികൾക്ക് പഠന പ്രക്രിയ കൂടുതൽ ആകർഷകമാകുകയും ആശയങ്ങൾ കൂടുതൽ ദീർഘകാലം മനസ്സിലാക്കപ്പെടുകയും ചെയ്യും. ക്ലാസിന് മുമ്പുള്ള കുറഞ്ഞ സമ്മർദ്ദമുള്ള ക്വിസ് ഒരു വിഷയത്തിൽ പഠന ഉദ്ദേശ്യം സജ്ജമാക്കുകയും, ക്ലാസിന് ശേഷം മറ്റൊരു ക്വിസ് കൂടുതൽ മനസ്സിലാക്കലിന് സഹായിക്കുകയും ചെയ്യും. ഈ പാഠ്യപദ്ധതി ലവചികവും രസകരവുമാണ്, മുഴുവനായോ ഭാഗികമായോ എടുക്കാവുന്നതാണ്. പ്രോജക്ടുകൾ ചെറിയതിൽ ആരംഭിച്ച് 12 ആഴ്ചകളുടെ ചക്രം അവസാനിക്കുമ്പോൾ കൂടുതൽ സങ്കീർണ്ണമാകും.

> **ക്വിസുകളെക്കുറിച്ചുള്ള കുറിപ്പ്**: എല്ലാ ക്വിസുകളും [ഈ ആപ്പിൽ](https://red-field-0a6ddfd03.1.azurestaticapps.net/) ഉൾപ്പെടുത്തിയിട്ടുണ്ട്, ഓരോന്നിലും മൂന്ന് ചോദ്യങ്ങളുള്ള 50 ക്വിസുകൾ. അവ പാഠങ്ങളിൽ നിന്ന് ലിങ്ക് ചെയ്തിട്ടുണ്ട്, പക്ഷേ ക്വിസ് ആപ്പ് ലോക്കലായി പ്രവർത്തിപ്പിക്കാം; `etc/quiz-app` ഫോൾഡറിൽ നിർദ്ദേശങ്ങൾ പാലിക്കുക.

## ഓഫ്‌ലൈൻ ആക്‌സസ്

[Docsify](https://docsify.js.org/#/) ഉപയോഗിച്ച് ഈ ഡോക്യുമെന്റേഷൻ ഓഫ്‌ലൈൻ പ്രവർത്തിപ്പിക്കാം. ഈ റീപ്പോ ഫോർക്ക് ചെയ്ത്, നിങ്ങളുടെ ലോക്കൽ മെഷീനിൽ [Docsify ഇൻസ്റ്റാൾ](https://docsify.js.org/#/quickstart) ചെയ്യുക, തുടർന്ന് ഈ റീപ്പോയുടെ റൂട്ട് ഫോൾഡറിൽ `docsify serve` ടൈപ്പ് ചെയ്യുക. വെബ്സൈറ്റ് നിങ്ങളുടെ ലോക്കൽഹോസ്റ്റിൽ പോർട്ട് 3000-ൽ ലഭ്യമായിരിക്കും: `localhost:3000`. പാഠ്യപദ്ധതിയുടെ PDF [ഈ ലിങ്കിൽ](../../../../../../../../etc/pdf/readme.pdf) ലഭ്യമാണ്.
