# ഈ പാഠ്യപദ്ധതിയുമായി ആരംഭിക്കുക

## നിങ്ങൾ ഒരു വിദ്യാർത്ഥിയാണോ?

താഴെ പറയുന്ന വിഭവങ്ങൾ ഉപയോഗിച്ച് ആരംഭിക്കുക:

* [വിദ്യാർത്ഥി ഹബ് പേജ്](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) ഈ പേജിൽ നിങ്ങൾക്ക് തുടക്കക്കാർക്കുള്ള വിഭവങ്ങൾ, വിദ്യാർത്ഥി പാക്കുകൾ, കൂടാതെ ഒരു സൗജന്യ സർട്ടിഫിക്കറ്റ് വൗച്ചർ നേടാനുള്ള മാർഗങ്ങൾ ലഭിക്കും. ഈ പേജ് ബുക്ക്മാർക്ക് ചെയ്ത് മാസത്തിൽ കുറഞ്ഞത് ഒരിക്കൽ പരിശോധിക്കുക, കാരണം ഞങ്ങൾ ഉള്ളടക്കം മാറ്റുന്നു.
* [Microsoft Student Learn ambassadors](https://studentambassadors.microsoft.com?WT.mc_id=academic-77998-cacaste) ഒരു ആഗോള വിദ്യാർത്ഥി അംബാസഡർ സമൂഹത്തിൽ ചേരുക, ഇത് Microsoft-ലേക്ക് പ്രവേശിക്കാനുള്ള നിങ്ങളുടെ മാർഗമായിരിക്കാം.

**വിദ്യാർത്ഥികൾ**, ഈ പാഠ്യപദ്ധതി ഉപയോഗിക്കാൻ രണ്ട് മാർഗങ്ങൾ ഉണ്ട്. ആദ്യം, നിങ്ങൾക്ക് GitHub-ൽ നേരിട്ട് ടെക്സ്റ്റും കോഡും വായിക്കാം. നിങ്ങൾക്ക് ഏതെങ്കിലും നോട്ട്‌ബുക്കുകളിൽ കോഡ് പ്രവർത്തിപ്പിക്കണമെങ്കിൽ - [ഞങ്ങളുടെ നിർദ്ദേശങ്ങൾ വായിക്കുക](./how-to-run.md), കൂടാതെ അത് എങ്ങനെ ചെയ്യാമെന്ന് കൂടുതൽ ഉപദേശം [ഈ ബ്ലോഗ് പോസ്റ്റിൽ](https://soshnikov.com/education/how-to-execute-notebooks-from-github/) കണ്ടെത്തുക.

> **Note**: [ഈ പാഠ്യപദ്ധതിയിലെ കോഡ് പ്രവർത്തിപ്പിക്കുന്നതിന് നിർദ്ദേശങ്ങൾ](./how-to-run.md)

## സ്വയം പഠനം

എന്നാൽ, നിങ്ങൾ ഈ കോഴ്സ് സ്വയം പഠന പ്രോജക്റ്റായി എടുക്കാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, ഞങ്ങൾ ഈ റീപ്പോ മുഴുവൻ നിങ്ങളുടെ GitHub അക്കൗണ്ടിലേക്ക് ഫോർക്ക് ചെയ്യാൻ, നിങ്ങളുടെ സ്വന്തം അല്ലെങ്കിൽ ഒരു ഗ്രൂപ്പിനൊപ്പം വ്യായാമങ്ങൾ പൂർത്തിയാക്കാൻ നിർദ്ദേശിക്കുന്നു:

* പ്രീ-ലക്ചർ ക്വിസ് ഉപയോഗിച്ച് ആരംഭിക്കുക.
* ലക്ചറിനുള്ള പ്രാരംഭ ടെക്സ്റ്റ് വായിക്കുക.
* ലക്ചറിന് അധിക നോട്ട്‌ബുക്കുകൾ ഉണ്ടെങ്കിൽ, അവ വായിച്ച് കോഡ് പ്രവർത്തിപ്പിക്കുക. TensorFlow, PyTorch നോട്ട്‌ബുക്കുകൾ രണ്ടും നൽകിയിട്ടുണ്ടെങ്കിൽ, നിങ്ങൾക്ക് ഇഷ്ടമുള്ള ഫ്രെയിംവർക്കിൽ ശ്രദ്ധ കേന്ദ്രീകരിക്കാം.
* നോട്ട്‌ബുക്കുകളിൽ കോഡ് കുറച്ച് മാറ്റി പരീക്ഷിക്കാൻ ആവശ്യമായ ചില ചലഞ്ചുകൾ അടങ്ങിയിരിക്കും.
* പോസ്റ്റ്-ലക്ചർ ക്വിസ് എടുക്കുക.
* മോഡ്യൂളിനോട് അനുബന്ധമായ ലാബ് ഉണ്ടെങ്കിൽ - അസൈൻമെന്റ് പൂർത്തിയാക്കുക.
* [ചർച്ചാ ബോർഡ്](https://github.com/microsoft/AI-For-Beginners/discussions) സന്ദർശിച്ച് "ശബ്ദമായി പഠിക്കുക".

> കൂടുതൽ പഠനത്തിനായി, ഈ [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) മോഡ്യൂളുകളും പഠന പാതകളും പിന്തുടരാൻ ഞങ്ങൾ ശുപാർശ ചെയ്യുന്നു.

**അധ്യാപകർ**, ഈ പാഠ്യപദ്ധതി എങ്ങനെ ഉപയോഗിക്കാമെന്ന് [ചില നിർദ്ദേശങ്ങൾ](./for-teachers.md) ഉൾപ്പെടുത്തിയിട്ടുണ്ട്.

## പാഠശാസ്ത്രം

ഈ പാഠ്യപദ്ധതി നിർമ്മിക്കുമ്പോൾ, ഞങ്ങൾ രണ്ട് പാഠശാസ്ത്ര തത്വങ്ങൾ തിരഞ്ഞെടുക്കുകയും അവ **പ്രോജക്റ്റ് അടിസ്ഥാനമാക്കിയുള്ള**തും **തുടർച്ചയായ ക്വിസുകൾ** ഉൾക്കൊള്ളുന്നതുമാണ്.

ഉള്ളടക്കം പ്രോജക്റ്റുകളുമായി പൊരുത്തപ്പെടുന്നുവെന്ന് ഉറപ്പാക്കുന്നതിലൂടെ, പ്രക്രിയ വിദ്യാർത്ഥികൾക്ക് കൂടുതൽ ആകർഷകമാകുകയും ആശയങ്ങളുടെ ഓർമ്മശക്തി വർദ്ധിപ്പിക്കുകയും ചെയ്യും. കൂടാതെ, ക്ലാസിന് മുമ്പുള്ള ഒരു ലോ-സ്റ്റേക്ക്‌സ് ക്വിസ് ഒരു വിഷയം പഠിക്കാനുള്ള വിദ്യാർത്ഥിയുടെ ഉദ്ദേശം സജ്ജമാക്കുന്നു, ക്ലാസിന് ശേഷം ഒരു രണ്ടാം ക്വിസ് കൂടുതൽ ഓർമ്മശക്തി ഉറപ്പാക്കുന്നു. ഈ പാഠ്യപദ്ധതി ഇളവുള്ളതും രസകരവുമാണ്, മുഴുവനായോ ഭാഗികമായോ എടുക്കാം. പ്രോജക്റ്റുകൾ ചെറിയതായും 12-ആഴ്ച ചക്രത്തിന്റെ അവസാനം കൂടുതൽ സങ്കീർണ്ണമായതായും ആരംഭിക്കുന്നു.

> **ക്വിസുകളെക്കുറിച്ചുള്ള ഒരു കുറിപ്പ്**: എല്ലാ ക്വിസുകളും [ഈ ആപ്പിൽ](https://red-field-0a6ddfd03.1.azurestaticapps.net/) അടങ്ങിയിരിക്കുന്നു, ഓരോന്നിലും മൂന്ന് ചോദ്യങ്ങളുള്ള 50 ക്വിസുകൾ. അവ പാഠങ്ങളിൽ നിന്ന് ലിങ്ക് ചെയ്തിരിക്കുന്നു, പക്ഷേ ക്വിസ് ആപ്പ് ലോക്കലായി പ്രവർത്തിപ്പിക്കാം; `etc/quiz-app` ഫോൾഡറിൽ ഉള്ള നിർദ്ദേശങ്ങൾ പിന്തുടരുക.

## ഓഫ്‌ലൈൻ ആക്സസ്

[Docsify](https://docsify.js.org/#/) ഉപയോഗിച്ച് ഈ ഡോക്യുമെന്റേഷൻ ഓഫ്‌ലൈൻ പ്രവർത്തിപ്പിക്കാം. ഈ റീപ്പോ ഫോർക്ക് ചെയ്യുക, [Docsify ഇൻസ്റ്റാൾ ചെയ്യുക](https://docsify.js.org/#/quickstart) നിങ്ങളുടെ ലോക്കൽ മെഷീനിൽ, തുടർന്ന് ഈ റീപ്പോയുടെ റൂട്ട് ഫോൾഡറിൽ `docsify serve` ടൈപ്പ് ചെയ്യുക. വെബ്സൈറ്റ് നിങ്ങളുടെ ലോക്കൽഹോസ്റ്റിൽ പോർട്ട് 3000-ൽ സർവ് ചെയ്യപ്പെടും: `localhost:3000`. പാഠ്യപദ്ധതിയുടെ ഒരു pdf [ഈ ലിങ്കിൽ](../../../../lessons/0-course-setup/etc/pdf/readme.pdf) ലഭ്യമാണ്.
