# സംഭാവനകൾ തേടുന്നു

നാം നിലവിൽ താഴെ പറയുന്ന വിഷയങ്ങളിൽ സജീവമായി സംഭാവനകൾ തേടുകയാണ്:

- [ ] ഡീപ്പ് റീഇൻഫോഴ്‌സ്‌മെന്റ് ലേണിങ്ങ് സംബന്ധിച്ച വിഭാഗം എഴുതുക
- [ ] ഒബ്ജക്റ്റ് ഡിറ്റക്ഷൻ വിഭാഗവും നോട്ട്‌ബുക്കും മെച്ചപ്പെടുത്തുക
- [ ] PyTorch Lightning (ഈ [വിഭാഗത്തിനായി](https://github.com/microsoft/AI-For-Beginners/blob/main/3-NeuralNetworks/05-Frameworks/README.md))
- [ ] നെയിംഡ് എന്റിറ്റി റെക്കഗ്നിഷൻ സംബന്ധിച്ച വിഭാഗവും സാമ്പിളുകളും എഴുതുക
- [ ] ഞങ്ങളുടെ സ്വന്തം എംബെഡിംഗുകൾ പരിശീലിപ്പിക്കാൻ സാമ്പിളുകൾ സൃഷ്ടിക്കുക ഈ [വിഭാഗത്തിനായി](https://github.com/microsoft/AI-For-Beginners/tree/main/5-NLP/15-LanguageModeling)
