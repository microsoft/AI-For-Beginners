# ಪೆಟ್ಸ್ ಮುಖಗಳ ವರ್ಗೀಕರಣ

[AI for Beginners Curriculum](https://github.com/microsoft/ai-for-beginners) ನಿಂದ ಪ್ರಯೋಗಾಲಯದ ಕಾರ್ಯ.

## ಕಾರ್ಯ

ನೀವು ಪೆಟ್ ನರ್ಸರಿ ಅಪ್ಲಿಕೇಶನ್ ಅಭಿವೃದ್ಧಿಪಡಿಸಬೇಕಾಗಿದೆ ಎಂದು ಕಲ್ಪಿಸಿ, ಎಲ್ಲ ಪೆಟ್ಸ್ ಅನ್ನು ಕ್ಯಾಟಲಾಗ್ ಮಾಡಲು. ಇಂತಹ ಅಪ್ಲಿಕೇಶನ್‌ನ ಒಂದು ಮಹತ್ವದ ವೈಶಿಷ್ಟ್ಯವೆಂದರೆ ಫೋಟೋದಿಂದ ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಜಾತಿಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು. ಇದನ್ನು ನ್ಯೂರಲ್ ನೆಟ್‌ವರ್ಕ್‌ಗಳನ್ನು ಬಳಸಿ ಯಶಸ್ವಿಯಾಗಿ ಮಾಡಬಹುದು.

ನೀವು **Pet Faces** ಡೇಟಾಸೆಟ್ ಬಳಸಿ ಬೆಕ್ಕು ಮತ್ತು ನಾಯಿ ವಿವಿಧ ಜಾತಿಗಳನ್ನು ವರ್ಗೀಕರಿಸಲು ಒಂದು ಕನ್ವಲ್ಯೂಷನಲ್ ನ್ಯೂರಲ್ ನೆಟ್‌ವರ್ಕ್ ತರಬೇತಿ ಮಾಡಬೇಕಾಗಿದೆ.

## ಡೇಟಾಸೆಟ್

ನಾವು [Oxford-IIIT Pet Dataset](https://www.robots.ox.ac.uk/~vgg/data/pets/) ಅನ್ನು ಬಳಸಲಿದ್ದೇವೆ, ಇದರಲ್ಲಿ 37 ವಿಭಿನ್ನ ನಾಯಿ ಮತ್ತು ಬೆಕ್ಕು ಜಾತಿಗಳ ಚಿತ್ರಗಳಿವೆ.

![ನಾವು ಕೈಗಾರಿಕೆ ಮಾಡಲಿರುವ ಡೇಟಾಸೆಟ್](../../../../../../translated_images/kn/data.50b2a9d5484bdbf0.webp)

ಡೇಟಾಸೆಟ್ ಡೌನ್‌ಲೋಡ್ ಮಾಡಲು, ಈ ಕೋಡ್ ಸ್ನಿಪೆಟ್ ಬಳಸಿ:

```python
!wget https://thor.robots.ox.ac.uk/~vgg/data/pets/images.tar.gz
!tar xfz images.tar.gz
!rm images.tar.gz
```

**ಗಮನಿಸಿ:** Oxford-IIIT Pet Dataset ಚಿತ್ರಗಳು ಫೈಲ್‌ಹೆಸರಿನ ಮೂಲಕ (ಉದಾ., `Abyssinian_1.jpg`, `Bengal_2.jpg`) ಸಂಘಟಿತವಾಗಿವೆ. ನೋಟ್ಬುಕ್‌ನಲ್ಲಿ ಈ ಚಿತ್ರಗಳನ್ನು ಜಾತಿ-ನಿರ್ದಿಷ್ಟ ಉಪಡೈರೆಕ್ಟರಿಗಳಲ್ಲಿ ಸಂಘಟಿಸಲು ಕೋಡ್ ಇದೆ, ಇದರಿಂದ ವರ್ಗೀಕರಣ ಸುಲಭವಾಗುತ್ತದೆ.

## ಪ್ರಾರಂಭಿಸುವ ನೋಟ್ಬುಕ್

ಪ್ರಯೋಗಾಲಯವನ್ನು ಪ್ರಾರಂಭಿಸಲು [PetFaces.ipynb](PetFaces.ipynb) ಅನ್ನು ತೆರೆಯಿರಿ

## ಪಡೆದ ಪಾಠ

ನೀವು ಶೂನ್ಯದಿಂದ ಚಿತ್ರ ವರ್ಗೀಕರಣದ ಸಾಪೇಕ್ಷವಾಗಿ ಸಂಕೀರ್ಣ ಸಮಸ್ಯೆಯನ್ನು ಪರಿಹರಿಸಿದ್ದೀರಿ! ಬಹಳಷ್ಟು ವರ್ಗಗಳಿದ್ದವು, ಆದರೂ ನೀವು ಸಮರ್ಥವಾದ ನಿಖರತೆಯನ್ನು ಪಡೆಯಲು ಸಾಧ್ಯವಾಯಿತು! ಟಾಪ್-k ನಿಖರತೆಯನ್ನು ಅಳೆಯುವುದು ಸಹ ಅರ್ಥಪೂರ್ಣ, ಏಕೆಂದರೆ ಕೆಲವು ವರ್ಗಗಳನ್ನು ಮಾನವರಿಗೂ ಸ್ಪಷ್ಟವಾಗಿ ವಿಭಿನ್ನವಲ್ಲದ ಕಾರಣ ಗೊಂದಲವಾಗಬಹುದು.
