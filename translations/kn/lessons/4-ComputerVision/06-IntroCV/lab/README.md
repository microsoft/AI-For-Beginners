# ಆಪ್ಟಿಕಲ್ ಫ್ಲೋ ಬಳಸಿ ಚಲನೆಗಳನ್ನು ಪತ್ತೆಹಚ್ಚುವುದು

[AI for Beginners Curriculum](https://aka.ms/ai-beginners) ನಿಂದ ಪ್ರಯೋಗಾಲಯ ಕಾರ್ಯ.

## ಕಾರ್ಯ

[ಈ ವೀಡಿಯೋ](../../../../../../lessons/4-ComputerVision/06-IntroCV/lab/palm-movement.mp4) ಯನ್ನು ಪರಿಗಣಿಸಿ, ಇದರಲ್ಲಿ ಒಬ್ಬ ವ್ಯಕ್ತಿಯ ಹಸ್ತದ ತೊಡೆಯು ಸ್ಥಿರ ಹಿನ್ನೆಲೆಯ ಮೇಲೆ ಎಡಕ್ಕೆ/ಬಲಕ್ಕೆ/ಮೇಲಕ್ಕೆ/ಕೆಳಗೆ ಚಲಿಸುತ್ತದೆ.

<img src="../../../../../../translated_images/kn/palm-movement.341495f0e9c47da3.webp" width="30%" alt="ಹಸ್ತದ ಚಲನೆ ಫ್ರೇಮ್"/>

**ನಿಮ್ಮ ಗುರಿ** ಆಪ್ಟಿಕಲ್ ಫ್ಲೋ ಬಳಸಿ ವೀಡಿಯೋದಲ್ಲಿ ಯಾವ ಭಾಗಗಳಲ್ಲಿ ಮೇಲಕ್ಕೆ/ಕೆಳಗೆ/ಎಡಕ್ಕೆ/ಬಲಕ್ಕೆ ಚಲನೆಗಳಿವೆ ಎಂದು ನಿರ್ಧರಿಸುವುದು.

**ವಿಸ್ತಾರ ಗುರಿ** ಚರ್ಮದ ಬಣ್ಣವನ್ನು ಬಳಸಿಕೊಂಡು ಹಸ್ತ ಅಥವಾ ಬೆರಳುಗಳ ಚಲನೆಗಳನ್ನು ನಿಜವಾಗಿಯೂ ಟ್ರ್ಯಾಕ್ ಮಾಡುವುದು, [ಈ ಬ್ಲಾಗ್ ಪೋಸ್ಟ್](https://dev.to/amarlearning/finger-detection-and-tracking-using-opencv-and-python-586m) ಅಥವಾ [ಇಲ್ಲಿ](http://www.benmeline.com/finger-tracking-with-opencv-and-python/) ವಿವರಿಸಿದಂತೆ.

## ಪ್ರಾರಂಭಿಸುವ ನೋಟ್ಬುಕ್

ಪ್ರಯೋಗಾಲಯವನ್ನು ಪ್ರಾರಂಭಿಸಲು [MovementDetection.ipynb](MovementDetection.ipynb) ಅನ್ನು ತೆರೆಯಿರಿ

## ತಿಳಿವಳಿಕೆ

ಕೆಲವೊಮ್ಮೆ, ಚಲನೆ ಪತ್ತೆಹಚ್ಚುವುದು ಅಥವಾ ಬೆರಳು ಮುಚ್ಚು ಪತ್ತೆಹಚ್ಚುವುದು ಹೀಗೆ ಸಾಪೇಕ್ಷವಾಗಿ ಸಂಕೀರ್ಣ ಕಾರ್ಯಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಕಂಪ್ಯೂಟರ್ ದೃಷ್ಟಿ ಮೂಲಕ ಪರಿಹರಿಸಬಹುದು. ಆದ್ದರಿಂದ, OpenCV ಮುಂತಾದ ಗ್ರಂಥಾಲಯಗಳು ಏನು ಮಾಡಬಹುದು ಎಂಬುದನ್ನು ತಿಳಿದುಕೊಳ್ಳುವುದು ಬಹಳ ಸಹಾಯಕ.
