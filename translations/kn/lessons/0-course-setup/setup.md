# ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಪ್ರಾರಂಭಿಸುವುದು

## ನೀವು ವಿದ್ಯಾರ್ಥಿಯೇ?

ಈ ಸಂಪತ್ತನ್ನು ಬಳಸಿಕೊಂಡು ಪ್ರಾರಂಭಿಸಿ:

* [ವಿದ್ಯಾರ್ಥಿ ಹಬ್ ಪುಟ](https://docs.microsoft.com/learn/student-hub?WT.mc_id=academic-77998-cacaste) ಈ ಪುಟದಲ್ಲಿ, ಪ್ರಾರಂಭಿಕ ಸಂಪತ್ತುಗಳು, ವಿದ್ಯಾರ್ಥಿ ಪ್ಯಾಕ್‌ಗಳು, ಮತ್ತು ಉಚಿತ ಪ್ರಮಾಣಪತ್ರ ವೌಚರ್ ಪಡೆಯುವ ಮಾರ್ಗಗಳನ್ನು ಕಾಣಬಹುದು. ಈ ಪುಟವನ್ನು ಬುಕ್‌ಮಾರ್ಕ್ ಮಾಡಿ ಮತ್ತು ಸಮಯದಿಂದ ಸಮಯಕ್ಕೆ ಪರಿಶೀಲಿಸಿ, ಏಕೆಂದರೆ ನಾವು ಕನಿಷ್ಠ ತಿಂಗಳಿಗೆ ಒಂದು ಬಾರಿ ವಿಷಯವನ್ನು ಬದಲಾಯಿಸುತ್ತೇವೆ.
* [Microsoft Student Learn Ambassadors](https://studentambassadors.microsoft.com?WT.mc_id=academic-77998-cacaste) ವಿದ್ಯಾರ್ಥಿ ಅಂಬಾಸಿಡರ್‌ಗಳ ಜಾಗತಿಕ ಸಮುದಾಯಕ್ಕೆ ಸೇರಿ, ಇದು Microsoft ಗೆ ನಿಮ್ಮ ಪ್ರವೇಶದ ಮಾರ್ಗವಾಗಬಹುದು.

**ವಿದ್ಯಾರ್ಥಿಗಳು**, ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಬಳಸಲು ಕೆಲವು ಮಾರ್ಗಗಳಿವೆ. ಮೊದಲನೆಯದಾಗಿ, ನೀವು ಕೇವಲ ಪಠ್ಯವನ್ನು ಓದಿ ಮತ್ತು GitHub ನಲ್ಲಿ ನೇರವಾಗಿ ಕೋಡ್ ಅನ್ನು ನೋಡಬಹುದು. ನೀವು ಯಾವುದೇ ನೋಟುಬುಕ್‌ಗಳಲ್ಲಿ ಕೋಡ್ ಅನ್ನು ಚಲಾಯಿಸಲು ಬಯಸಿದರೆ - [ನಮ್ಮ ಸೂಚನೆಗಳನ್ನು ಓದಿ](./how-to-run.md), ಮತ್ತು ಅದನ್ನು ಹೇಗೆ ಮಾಡುವುದು ಎಂಬುದರ ಬಗ್ಗೆ ಹೆಚ್ಚಿನ ಸಲಹೆಗಳನ್ನು [ಈ ಬ್ಲಾಗ್ ಪೋಸ್ಟ್‌ನಲ್ಲಿ](https://soshnikov.com/education/how-to-execute-notebooks-from-github/) ಹುಡುಕಿ.

> **Note**: [ಈ ಪಠ್ಯಕ್ರಮದಲ್ಲಿ ಕೋಡ್ ಅನ್ನು ಹೇಗೆ ಚಲಾಯಿಸಬೇಕು ಎಂಬ ಸೂಚನೆಗಳು](./how-to-run.md)

## ಸ್ವಯಂ ಅಧ್ಯಯನ

ಆದರೆ, ನೀವು ಈ ಕೋರ್ಸ್ ಅನ್ನು ಸ್ವಯಂ ಅಧ್ಯಯನ ಯೋಜನೆಯಾಗಿ ತೆಗೆದುಕೊಳ್ಳಲು ಬಯಸಿದರೆ, ನಾವು ಸಂಪೂರ್ಣ ರೆಪೊವನ್ನು ನಿಮ್ಮ GitHub ಖಾತೆಗೆ ಫೋರ್ಕ್ ಮಾಡಿ, ವ್ಯಾಯಾಮಗಳನ್ನು ನಿಮ್ಮದೇ ಆದ ಅಥವಾ ಗುಂಪಿನೊಂದಿಗೆ ಪೂರ್ಣಗೊಳಿಸಲು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ:

* ಪಠ್ಯದ ಮುಂಚಿನ ಕ್ವಿಜ್‌ನಿಂದ ಪ್ರಾರಂಭಿಸಿ.
* ಪಠ್ಯದ ಪರಿಚಯ ಪಠ್ಯವನ್ನು ಓದಿ.
* ಪಠ್ಯದಲ್ಲಿ ಹೆಚ್ಚುವರಿ ನೋಟುಬುಕ್‌ಗಳು ಇದ್ದರೆ, ಅವುಗಳನ್ನು ಓದಿ ಮತ್ತು ಕೋಡ್ ಅನ್ನು ಚಲಾಯಿಸಿ. TensorFlow ಮತ್ತು PyTorch ನೋಟುಬುಕ್‌ಗಳು ಒದಗಿಸಿದರೆ, ನೀವು ಅವುಗಳಲ್ಲಿ ಒಂದರ ಮೇಲೆ ಗಮನಹರಿಸಬಹುದು - ನಿಮ್ಮ ಮೆಚ್ಚಿನ ಫ್ರೇಮ್‌ವರ್ಕ್ ಆಯ್ಕೆಮಾಡಿ.
* ನೋಟುಬುಕ್‌ಗಳಲ್ಲಿ ಸಾಮಾನ್ಯವಾಗಿ ಕೆಲವು ಸವಾಲುಗಳು ಇರುತ್ತವೆ, ಅವುಗಳನ್ನು ಪ್ರಯೋಗಿಸಲು ಕೋಡ್ ಅನ್ನು ಸ್ವಲ್ಪ ಬದಲಾಯಿಸಲು ಅಗತ್ಯವಿರುತ್ತದೆ.
* ಪಠ್ಯದ ನಂತರದ ಕ್ವಿಜ್ ತೆಗೆದುಕೊಳ್ಳಿ.
* ಮೋಡ್ಯೂಲ್‌ಗೆ ಲ್ಯಾಬ್ ಲಗತ್ತಿಸಿದರೆ - ಅಸೈನ್‌ಮೆಂಟ್ ಅನ್ನು ಪೂರ್ಣಗೊಳಿಸಿ.
* [ಚರ್ಚಾ ಮಂಡಳಿ](https://github.com/microsoft/AI-For-Beginners/discussions) ಗೆ ಭೇಟಿ ನೀಡಿ "ತಮ್ಮ ಕಲಿಕೆಯನ್ನು ಹಂಚಿಕೊಳ್ಳಲು".

> ಹೆಚ್ಚಿನ ಅಧ್ಯಯನಕ್ಕಾಗಿ, ಈ [Microsoft Learn](https://docs.microsoft.com/en-us/users/dmitrysoshnikov-9132/collections/31zgizg2p418yo/?WT.mc_id=academic-77998-cacaste) ಮೋಡ್ಯೂಲ್‌ಗಳು ಮತ್ತು ಕಲಿಕೆ ಮಾರ್ಗಗಳನ್ನು ಅನುಸರಿಸಲು ಶಿಫಾರಸು ಮಾಡುತ್ತೇವೆ.

**ಶಿಕ್ಷಕರು**, ನಾವು ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಹೇಗೆ ಬಳಸಬೇಕು ಎಂಬುದರ ಬಗ್ಗೆ [ಕೆಲವು ಸಲಹೆಗಳನ್ನು ಸೇರಿಸಿದ್ದೇವೆ](./for-teachers.md).

## ಪೆಡಗೋಗಿ

ನಾವು ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ನಿರ್ಮಿಸುವಾಗ ಎರಡು ಪೆಡಗೋಗಿಕ ತತ್ವಗಳನ್ನು ಆಯ್ಕೆ ಮಾಡಿದ್ದೇವೆ: ಇದು ಕೈಗೊಳ್ಳುವ **ಪ್ರಾಜೆಕ್ಟ್ ಆಧಾರಿತ** ಮತ್ತು ಇದು **ನಿಯಮಿತ ಕ್ವಿಜ್‌ಗಳನ್ನು** ಒಳಗೊಂಡಿರುತ್ತದೆ ಎಂಬುದನ್ನು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳುವುದು.

ಪಠ್ಯವು ಪ್ರಾಜೆಕ್ಟ್‌ಗಳಿಗೆ ಹೊಂದಾಣಿಕೆಯಾಗುವಂತೆ ಖಚಿತಪಡಿಸುವ ಮೂಲಕ, ಪ್ರಕ್ರಿಯೆಯನ್ನು ವಿದ್ಯಾರ್ಥಿಗಳಿಗೆ ಹೆಚ್ಚು ಆಕರ್ಷಕವಾಗಿಸುತ್ತದೆ ಮತ್ತು ಪರಿಕಲ್ಪನೆಗಳ ಸಂಗ್ರಹವನ್ನು ಹೆಚ್ಚಿಸುತ್ತದೆ. ಜೊತೆಗೆ, ತರಗತಿಗೆ ಮುಂಚಿನ ಕಡಿಮೆ-ಮೌಲ್ಯದ ಕ್ವಿಜ್ ವಿದ್ಯಾರ್ಥಿಯ ಉದ್ದೇಶವನ್ನು ವಿಷಯವನ್ನು ಕಲಿಯುವತ್ತ ಸೆಳೆಯುತ್ತದೆ, ಮತ್ತು ತರಗತಿಯ ನಂತರದ ಎರಡನೇ ಕ್ವಿಜ್ ಹೆಚ್ಚಿನ ಸಂಗ್ರಹವನ್ನು ಖಚಿತಪಡಿಸುತ್ತದೆ. ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಲಚೀಲ ಮತ್ತು ಮನರಂಜನೆಯಿಂದ ಕೂಡಿದಂತೆ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ ಮತ್ತು ಸಂಪೂರ್ಣವಾಗಿ ಅಥವಾ ಭಾಗಶಃ ತೆಗೆದುಕೊಳ್ಳಬಹುದು. ಪ್ರಾಜೆಕ್ಟ್‌ಗಳು ಚಿಕ್ಕದಾಗಿ ಪ್ರಾರಂಭವಾಗುತ್ತವೆ ಮತ್ತು 12-ವಾರದ ಚಕ್ರದ ಕೊನೆಯಲ್ಲಿ ಕ್ರಮೇಣ ಸಂಕೀರ್ಣವಾಗುತ್ತವೆ.

> **ಕ್ವಿಜ್‌ಗಳ ಬಗ್ಗೆ ಒಂದು ಟಿಪ್ಪಣಿ**: ಎಲ್ಲಾ ಕ್ವಿಜ್‌ಗಳು [ಈ ಅಪ್ಲಿಕೇಶನ್‌ನಲ್ಲಿ](https://red-field-0a6ddfd03.1.azurestaticapps.net/) ಒಳಗೊಂಡಿವೆ, ಒಟ್ಟು 50 ಕ್ವಿಜ್‌ಗಳು, ಪ್ರತಿ ಕ್ವಿಜ್‌ಗೆ ಮೂರು ಪ್ರಶ್ನೆಗಳನ್ನು ಹೊಂದಿವೆ. ಅವು ಪಾಠಗಳಲ್ಲಿ ಲಿಂಕ್ ಮಾಡಲ್ಪಟ್ಟಿವೆ ಆದರೆ ಕ್ವಿಜ್ ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಸ್ಥಳೀಯವಾಗಿ ಚಲಾಯಿಸಬಹುದು; `etc/quiz-app` ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಸೂಚನೆಗಳನ್ನು ಅನುಸರಿಸಿ.

## ಆಫ್‌ಲೈನ್ ಪ್ರವೇಶ

ನೀವು [Docsify](https://docsify.js.org/#/) ಬಳಸಿ ಈ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಅನ್ನು ಆಫ್‌ಲೈನ್‌ನಲ್ಲಿ ಚಲಾಯಿಸಬಹುದು. ಈ ರೆಪೊವನ್ನು ಫೋರ್ಕ್ ಮಾಡಿ, [Docsify ಅನ್ನು ಸ್ಥಾಪಿಸಿ](https://docsify.js.org/#/quickstart) ನಿಮ್ಮ ಸ್ಥಳೀಯ ಯಂತ್ರದಲ್ಲಿ, ನಂತರ ಈ ರೆಪೊದ ಮೂಲ ಫೋಲ್ಡರ್‌ನಲ್ಲಿ `docsify serve` ಟೈಪ್ ಮಾಡಿ. ವೆಬ್‌ಸೈಟ್ ನಿಮ್ಮ ಸ್ಥಳೀಯಹೋಸ್ಟ್‌ನಲ್ಲಿ ಪೋರ್ಟ್ 3000 ನಲ್ಲಿ ಚಲಾಯಿಸಲಾಗುತ್ತದೆ: `localhost:3000`. ಪಠ್ಯಕ್ರಮದ ಪಿಡಿಎಫ್ [ಈ ಲಿಂಕ್‌ನಲ್ಲಿ](../../../../lessons/0-course-setup/etc/pdf/readme.pdf) ಲಭ್ಯವಿದೆ.
