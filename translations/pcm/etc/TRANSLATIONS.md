# Help us translate di lessons

We go happy if you fit help us translate di lessons wey dey dis curriculum!

## Rules

Inside each lesson folder and lesson introduction folder, you go see folders wey get di translated markdown files.

> Abeg, no translate any code wey dey inside di code sample files; di only tins wey you go translate na README, assignments, and di quizzes. Thank you!

Make sure say di translated files follow dis naming style:

**README._[language]_.md**

Where _[language]_ na two-letter language abbreviation wey follow di ISO 639-1 standard (e.g. `README.es.md` for Spanish and `README.nl.md` for Dutch).

**assignment._[language]_.md**

Just like README, abeg translate di assignments too.

**Quizzes**

1. Put your translation for di quiz-app by adding file for here: https://github.com/microsoft/AI-For-Beginners/tree/main/etc/quiz-app/src/assets/translations, make sure say you use correct naming style (en.json, fr.json). **Abeg no change di words 'true' or 'false'. Thank you!**

2. Add your language code for di dropdown wey dey di quiz-app's App.vue file.

3. Edit di quiz-app's [translations index.js file](https://github.com/microsoft/AI-For-Beginners/blob/main/etc/quiz-app/src/assets/translations/index.js) to add your language.

4. Last last, edit ALL di quiz links wey dey your translated README.md files so dem go point directly to your translated quiz: https://red-field-0a6ddfd03.1.azurestaticapps.net/quiz/1 go turn https://red-field-0a6ddfd03.1.azurestaticapps.net/quiz/1?loc=id

**THANK YOU**

We really appreciate di work wey you dey do!
