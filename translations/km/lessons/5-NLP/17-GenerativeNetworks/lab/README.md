# ការបង្កើតអត្ថបទផ្ទាល់កម្រិតពាក្យដោយប្រើ RNNs

ការប្រលងបន្ទប់មួយពី [មេរៀន AI សម្រាប់អ្នកចាប់ផ្តើម](https://github.com/microsoft/ai-for-beginners)។

## បេសកកម្ម

ក្នុងបន្ទប់មួយនេះ អ្នអ្នកត្រូវយកសៀវភៅណាមួយ មួយ ហើយប្រើវាជាគំរូទិន្នន័យដើម្បីបណ្តុះបណ្តាលកម្មវិធីបង្កើតអត្ថបទផ្ទាល់កម្រិតពាក្យ។

## គំរូទិន្នន័យ

អ្នកអាចប្រើសៀវភៅណាមួយបានសូមស្វាគមន៍។ អ្នកអាចរកឃើញអត្ថបទឥតគិតថ្លៃច្រើននៅ [Project Gutenberg](https://www.gutenberg.org/), ជាឧទាហរណ៍ នេះគឺជាលីងផ្ទាល់ដល់ [Alice's Adventures in Wonderland](https://www.gutenberg.org/files/11/11-0.txt)) ដែលសរសេរដោយ Lewis Carroll។
