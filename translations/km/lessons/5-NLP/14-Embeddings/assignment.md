# កិច្ចកម្ម៖ ប្រដាប់កំណត់កំណត់ត្រា

ប្រើប្រាស់ប្រដាប់កំណត់កំណត់ត្រាដែលពាក់ព័ន្ធនឹងមេរៀននេះ (ចំពោះកំណែ PyTorch ឬ TensorFlow) ធ្វើការចាប់ផ្តើមបញ្ចប់វាឡើងវិញដោយប្រើឯកសារទិន្នន័យផ្ទាល់ខ្លួនរបស់អ្នក ប្រហែលមួយនៅពី Kaggle ដោយមានការផ្តល់ឱ្យឈ្មោះតាមសិទ្ធិ។ សរសេរ ប្រដាប់កំណត់កំណត់ត្រាឡើងវិញ ដើម្បីបំភ្លឺលើការស្វែងរកផ្ទាល់ខ្លួនរបស់អ្នក។ ព្យាយាមប្រើប្រាស់ប្រភេទទិន្នន័យផ្សេងទៀត ហើយចាប់យកការស្វែងរករបស់អ្នក ដោយប្រើអត្ថបទដូចជា [បទចំរៀង Beatles ទាំងនេះ](https://www.kaggle.com/datasets/jenlooper/beatles-lyrics)។
