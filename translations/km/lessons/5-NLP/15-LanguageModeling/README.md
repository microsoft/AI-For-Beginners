# бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶

бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһҮбһ¶ semantic embeddings бһҠбһјбһ…бһҮбһ¶ Word2Vec бһ“бһ·бһ„ GloVe бһӮбһәбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҠбҹҶбһ”бһјбһ„бһ‘бҹ…бһҡбһҖ **бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶** - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһҷбҹүбһ¶бһ„бһҺбһ¶бһҳбһҪбһҷбһўбһ¶бһ… *бһҷбһӣбҹӢ* (бһ¬ *бһҸбҹҶбһҺбһ¶бһ„*) бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһ“бҹғбһ—бһ¶бһҹбһ¶бҹ”

## [бһҹбҹҶбһҺбһҪбһҡбһ”бҹ’бһҡбһӣбһ„бһҳбһ»бһ“бһҳбҹүбҹ„бһ„бһ”бһ„бҹ’бһҡбҹҖбһ“](https://ff-quizzes.netlify.app/en/ai/quiz/29)

бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһӮбһәбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ“бҹ…бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖ бһҮбһ¶бһҡбһ”бҹҖбһ”бһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бҹ” бһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ–бҹ’бһҡбҹ„бҹҮбһҷбһҫбһ„бһҳбһ¶бһ“бһ…бҹҶбһ“бһҪбһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһ…бҹ’бһҡбһҫбһ“ бһҒбһҺбҹҲбһҠбҹӮбһӣбһ…бҹҶбһ“бһҪбһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһ¶бһ“бһҹбҹ’бһӣбһ¶бһҖбһҸбҹӮбһ„бһҸбҹӮбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҖбҹҶбһӣбһ¶бҹҶбһ„бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһ…бҹҶбһҺбһ¶бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһӣбһ¶бһҖбҹ” бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһ–бһҪбһҖбһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбһ¶бһ… **бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһ–бһ¶бһҖбҹ’бһҷбһҠбҹӮбһӣбһўбһ¶бһ”бһ¶бһ“** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӣбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ…бҹғбһҠбһ“бҹ’бһҷбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһңбһ¶бһҮбһ¶бһӮбҹҶбһҡбһјбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ”

## бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ Embeddings

бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ»бһ“бҹ—бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҷбһҫбһ„бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ semantic embeddings бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһҪбһ…бһ бһҫбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһңбһ¶бһӮбһҪбһҡбһІбҹ’бһҷбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһҗбһ¶бһҸбһҫ embeddings бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бҹ” бһҳбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бҹ–

* **N-Gram** бһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶ бһ–бҹҒбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ token бһҳбһҪбһҷбһҠбҹ„бһҷбһҳбһҫбһӣбһ‘бҹ… N tokens бһҳбһ»бһ“ (N-gram)
* **Continuous Bag-of-Words** (CBoW) бһ–бҹҒбһӣбһҠбҹӮбһӣбһҷбһҫбһ„бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ token бһҖбһҺбҹ’бһҸбһ¶бһӣ $W_0$ бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢ token $W_{-N}$, ..., $W_N$бҹ”
* **Skip-gram**, бһҠбҹӮбһӣбһҷбһҫбһ„бһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһҖбҹ’бһҡбһ»бһҳ token бһҮбһ¶бһ”бҹӢбһҒбҹ’бһ“бҹ… {$W_{-N},\dots, W_{-1}, W_1,\dots, W_N$} бһ–бһё token бһҖбһҺбҹ’бһҸбһ¶бһӣ $W_0$бҹ”

> бһҡбһјбһ”бһ—бһ¶бһ–бһ–бһё [бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮ](https://arxiv.org/pdf/1301.3781.pdf)

## вңҚпёҸ бһҹбҹҖбһңбһ—бҹ…бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ CBoW

бһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ…бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

* [бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ CBoW Word2Vec бһҮбһ¶бһҳбһҪбһҷ PyTorch](CBoW-PyTorch.ipynb)


## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“ бһҷбһҫбһ„бһ”бһ¶бһ“бһғбһҫбһүбһҗбһ¶ word embeddings бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбһјбһ…бһҮбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹҒбһҠбһ№бһҖбһ“бһ¶бҹҶ! бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһҷбһҫбһ„бһҠбһ№бһ„бһҗбһ¶бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ word embeddings бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҺбһ¶бһҹбҹӢбһ‘бҹҒ бһ бһҫбһҷбһҷбһҫбһ„бһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ word embeddings бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҒбҹ’бһӣбҹҮ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бҹ”

## [бһҹбҹҶбһҺбһҪбһҡбһ”бҹ’бһҡбһӣбһ„бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбҹүбҹ„бһ„бһ”бһ„бҹ’бһҡбҹҖбһ“](https://ff-quizzes.netlify.app/en/ai/quiz/30)

## бһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһЎбһҫбһ„бһңбһ·бһү

* [бһҳбҹҒбһҡбҹҖбһ“ PyTorch бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶](https://pytorch.org/tutorials/beginner/nlp/word_embeddings_tutorial.html).
* [бһҳбҹҒбһҡбҹҖбһ“ TensorFlow бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ Word2Vec](https://www.TensorFlow.org/tutorials/text/word2vec).
* бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ **gensim** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ embeddings бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҲбһҡбһҒбҹ’бһӣбһёбҹ— бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ‘бҹ… [бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮ](https://pytorch.org/tutorials/beginner/nlp/word_embeddings_tutorial.html).

## рҹҡҖ [бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҮбҹ– бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ Skip-Gram](lab/README.md)

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ бһҷбһҫбһ„бһҶбҹ’бһӣбһҫбһҷбһ”бһүбҹ’бһ бһ¶бһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбһјбһҠбһ–бһёбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҠбҹӮбһӣ skip-gram бһҮбҹҶбһ“бһҪбһҹ CBoWбҹ” [бһўбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ](lab/README.md)
