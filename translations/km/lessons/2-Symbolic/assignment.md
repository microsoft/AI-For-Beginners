# សាងសង់អង់តូឡូജി

ការសាងសង់មូលដ្ឋានចំណេះដឹងគឺពាក់ព័ន្ធទៅនឹងការបែងចែកម៉ូដែលដែលតំណាងឱ្យកត្តាអំពីប្រធានបទមួយ។ ជ្រើសរើសប្រធានបទមួយ - ដូចជាមនុស្សម្នាក់ ទីកន្លែងមួយ ឬរបស់មួយ - ហើយបង្កើតម៉ូដែលនៃប្រធានបទនោះ។ ប្រើបច្ចេកទេសមួយចំនួន និងយុទ្ធសាស្ត្រសាងសង់ម៉ូដែលដែលបានពិពណ៌នាក្រោមមេរៀននេះ។ ឧទាហរណ៍ គឺការបង្កើតអង់តូឡូជីនៃបន្ទប់រស់នៅមានសម្ភារៈផ្ទះ ម៉ូលចង្កៀង និងផ្សេងទៀត។ តើបន្ទប់រស់នៅខុសពីផ្ទះបាយយ៉ាងដូចម្តេច? បន្ទប់ទឹក? តើអ្នកដឹងដូចម្តេចថាវាជាបន្ទប់រស់នៅ មិនមែនបន្ទប់បរិច្ឆេទ? ប្រើ [Protégé](https://protege.stanford.edu/) ដើម្បីសាងសង់អង់តូឡូជីរបស់អ្នក។
