# бһҡбҹҖбһ“бһ”бһ„бҹ’бһҡбҹҖбһ“бһЎбһ¶бһ“бһ—бҹ’бһ“бҹҶбһІбҹ’бһҷбһӮбҹҒбһ…бһ”бһ¶бһ“бһ…бҹҒбһү

бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҳбһ“бҹ’бһ‘бһёбһҡвҖӢбһҹбһ·бһҖбҹ’бһҹбһ¶вҖӢбһ–бһё [AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Curriculum](https://github.com/microsoft/ai-for-beginners)бҹ”

## бһ”бҹҶбһҺбһ„бһҖбһ¶бһҡ

бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһәбһ”бһ„бҹ’бһҡбҹҖбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ”бҹҖбһ”бһҡбҹҖбһ“ (RL) бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ [бһЎбһ¶бһ“бһ—бҹ’бһ“бҹҶ](https://www.gymlibrary.ml/environments/classic_control/mountain_car/) бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ OpenAIбҹ”

## бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһЎбһ¶бһ“бһ—бҹ’бһ“бҹҶбһҳбһ¶бһ“бһЎбһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ…бһ„бһҒбҹ’бһҹбҹӮбһҖбҹ’бһ“бһ»бһ„бһңбһ¶бһӣбһ—бҹ’бһ“бҹҶбҹ” бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһәбһҸбҹ’бһҡбһјбһңбһӣбҹ„бһҸбһ…бҹҒбһүбһ–бһёбһңбһ¶бһӣбһ—бҹ’бһ“бҹҶбһ“бһ·бһ„бһҲбһ¶бһ“бһҠбһӣбҹӢбһ‘бһ„бҹӢбһҮбһ¶бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһ—бҹ’бһ“бҹҶбҹ” бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҡбһҪбһҳбһҳбһ¶бһ“ бһҖбҹҶбһ…бһ¶бһҸбҹӢбһЎбһ¶бһ“бһ‘бҹ…бһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„ бһҒбһ¶бһ„бһҹбҹ’бһҸбһ¶бҹҶ бһ¬бһҳбһ·бһ“бһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһүбһ‘бһёбһҸбһ¶бҹҶбһ„бһЎбһ¶бһ“бһҸбһ¶бһҳбһўбҹҗбһҖбҹ’бһҹ x бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бҹ”

## бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҠбҹҶбһ”бһјбһ„

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӣбҹҶбһ бһ¶бһҸбҹӢбһҠбҹ„бһҷбһ”бһҫбһҖ [MountainCar.ipynb](MountainCar.ipynb)

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҡбҹҖбһ“бһ”бһ¶бһ“

бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһҡбҹҖбһ“бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҡбҹҖбһ“бһ“бҹҒбҹҮбһҗбһ¶ бһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бҹ’бһҡбһҫбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбҹҢ RL бһ‘бҹ…бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҗбҹ’бһҳбһё бһҮбһ¶бһҖбһ¶бһҡбҹүбһәбһҸбҹ’бһ”бһ·бһҸбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһҮбһ¶бһ–бһ·бһҹбҹҒбһҹбһҠбҹ„бһҷбһҹбһ¶бһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ OpenAI Gym бһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһ бһҫбһҷбһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбһҳбһ·бһ“бһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫбһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһ“бҹғбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶбһҖбһјбһҠ Python бһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһ‘бҹҒбһҡбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҺбһ¶бһҳбһҪбһҷбһ‘бҹ…бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһҡбһёбһ’бһҳбҹҚ RL бһҮбһ¶бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ”
