អាល់ហ្គារីធម៍ចំលែកផ្អែកលើគំនិតដូចខាងក្រោម៖

* ដំណោះស្រាយត្រឹមត្រូវសម្រាប់បញ្ហាអាចត្រូវតំណាងជា **កោសិការី**
* **ការបន្តិចបន្តួច** អនុញ្ញាតឱ្យយើងបញ្ចូលរួមពីរដំណោះស្រាយមួយគ្នាដើម្បីទទួលបានដំណោះស្រាយថ្មីត្រឹមត្រូវ
* **ការជ្រើសរើស** ត្រូវបានប្រើដើម្បីជ្រើសយកដំណោះស្រាយល្អជាងដោយប្រើ **មុខងារសមត្ថភាព**
* **ការបម្លែង** ត្រូវបានបញ្ចូល ដើម្បីធ្វើឲ្យការបង្កើតបញ្ហាមិនមានសុវត្ថិភាព និងធ្វើឲ្យយើងចេញពីតម្លៃតំបន់តិចបំផុតក្នុងមូលដ្ឋាន

បើអ្នកចង់អនុវត្តអាល់ហ្គារីធម៍ចំលែក អ្នកត្រូវការដូចខាងក្រោម៖

 * រកវិធីសាស្រ្តកូដដំណោះស្រាយបញ្ហារបស់យើងដោយប្រើ **កោសិការី** g&in;&Gamma;
 * លើសំណុំកោសិការី &Gamma; ត្រូវកំណត់ **មុខងារសមត្ថភាព** fit: &Gamma;&rightarrow;**R**។ តម្លៃមុខងារតិចជាងបង្ហាញដំណោះស្រាយល្អជាង។
