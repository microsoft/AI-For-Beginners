# бһ”бһҺбҹ’бһҠбһ¶бһүбһ–бһ бһ»вҖӢбһҳбҹүбһјбһҠбҹҒбһӣ

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҮбҹ„бһӮбһҮбҹҗбһҷбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣ transformer бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ NLP бһҖбһҹбһ¶бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ¬бһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһ”бһүбҹ’бһ бһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһҳбһ¶бһ“бһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҖбһҫбһ“бһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ“бһ№бһ„ *бһ”бһүбҹ’бһ…бһјбһӣ* бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһҫбһӣбһғбһҫбһү бһ“бһ·бһ„бһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бҹ” бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһҖбһ·бһ…бҹ’бһ…бһҒбһ·бһҸбһҒбҹҶбһ”бҹ’бһҡбһ№бһ„бһ”бҹ’бһҡбҹӮбһ„бһҠбһјбһ…бһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһҠбҹ„бһҷ OpenAI бһ бһҫбһҷбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҗбһ¶ CLIP бһ“бһ·бһ„ DALL.Eбҹ”

## бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„ (CLIP)

бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғ CLIP бһӮбһәбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһ¶бһҳбһҪбһҷбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһҸбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ бһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбҹ”

бһҳбҹүбһјбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһёбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ–бһҺбҹҢбһ“бһ¶бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһўбһ”бҹӢбһ“бһёбһҳбһҪбһҷбҹ— бһҷбһҫбһ„бһҷбһҖбһӮбһј N бһ“бҹғ (бһҡбһјбһ”бһ—бһ¶бһ–, бһўбһҸбҹ’бһҗбһ”бһ‘) бһ бһҫбһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҮбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһңбҹүбһ·бһ…бһ‘бҹҗбһҡ I<sub>1</sub>,..., I<sub>N</sub> / T<sub>1</sub>, ..., T<sub>N</sub>бҹ” бһҸбҹҶбһҺбһ¶бһ„бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһӮбҹ’бһ“бһ¶бҹ” бһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҖбһ¶бһҡбһҹбҹ’бһҡбһҠбҹҖбһ„ cosine бһҡбһңбһ¶бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһҠбҹӮбһӣбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһӮбҹ’бһ“бһ¶ (бһ§. I<sub>i</sub> бһ“бһ·бһ„ T<sub>i</sub>) бһ бһҫбһҷбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһҹбҹ’бһҡбһҠбҹҖбһ„ cosine бһҡбһңбһ¶бһ„бһӮбһјбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһ“бҹҒбҹҮбһ бһҫбһҷбһҮбһ¶бһ бҹҒбһҸбһ»бһ•бһӣбһҠбҹӮбһӣбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһ бҹ…бһҗбһ¶ **бһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһӮбҹ’бһ“бһ¶**бҹ”

бһҳбҹүбһјбһҠбҹӮбһӣ/бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ CLIP бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ“бҹ… [OpenAI GitHub](https://github.com/openai/CLIP)бҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ“бҹ… [бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бҹ’бһӣбһ»бһҖбһ“бҹҒбҹҮ](https://openai.com/blog/clip/), бһ“бһ·бһ„бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ… [бһҜбһҖбһҹбһ¶бһҡвҖӢбһ“бҹҒбҹҮ](https://arxiv.org/pdf/2103.00020.pdf)бҹ”

бһ–бҹҒбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҡбһҪбһ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһІбҹ’бһҷбһңбһ¶бһ”бҹ’бһҡбһўбһ”бҹӢбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбһҳбҹ’бһҡбһјбһң бһ бһҫбһҷбһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһҳбһҖбһҮбһ¶бһҸбһ„бҹӢбһҹбҹҗбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ—бһ¶бһ–бҹ” CLIP бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһ…бҹ’бһҡбһҫбһ“бҹ–

**бһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–**

бһҷбһҫбһ„бһҷбһҖбһӮбҹҶбһҡбһјбһҷбһӣбҹӢбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳ бһ§. бһҖбҹ’бһҡбһ»бһҳбһҶбҹ’бһҳбһ¶, бһҹбһҸбҹ’бһңвҖӢбһҶбҹ’бһҖбҹӮ бһ“бһ·бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһҪбһҷ бһҠбҹ„бһҷбһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ”бһ‘бһҸбһҳбҹ’бһҡбһјбһң бһ…бҹ’бһҡбһҫбһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢбҹ– "*бһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹғбһҶбҹ’бһҳбһ¶бһҳбһҪбһҷ*", "*бһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹғбһҶбҹ’бһҖбҹӮбһҳбһҪбһҷ*", "*бһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹғбһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢ*бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ—бһ¶бһ– 3 бһҷбһҫбһ„бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ–бһҺбҹҢбһҹбһ¶бһ“бҹӢвҖӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

бһ”бһҫбһҖбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ [Clip.ipynb](Clip.ipynb) бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣ CLIP бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

## бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷ VQGAN+ CLIP

CLIP бһҖбҹҸбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһёбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбһҳбҹ’бһҡбһјбһңбһ”бһ¶бһ“бҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҳбҹүбһјбһҠбҹӮбһӣ **бһҮбһ¶бһҹбҹ’бһҡбҹ’бһҸбһёбһҳбһҪбһҷ** бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһӣбһҫбһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бһјбһӣбһ”someбҹ” бһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһәбһ бҹ…бһҗбһ¶ [VQGAN](https://compvis.github.io/taming-transformers/) (Vector-Quantized GAN)бҹ”

бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢ VQGAN бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ–бһё [GAN](../../4-ComputerVision/10-GANs/README.md) бһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“бҹ–
* бһ”бҹ’бһҡбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ transformer autoregressive бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһҠбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһҡбһҹбҹ’бһҸбҹ’бһҡ context-rich бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһ•бҹ’бһ“бҹӮбһҖбһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһҡбһҹбҹ’бһҸбҹ’бһҡбһҒбһ¶бһ„бһӣбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ“бһҮбһ¶бһҳбһҪбһҷ [CNN](../../4-ComputerVision/07-ConvNets/README.md)
* бһ”бҹ’бһҡбһҫбһҮбһ¶бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҸбһјбһ… (sub-image discriminator) бһҠбҹӮбһӣбһҡбһҖбһғбһҫбһүбһҗбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ“бҹғбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ¶бһ“ "бһ–бһ·бһҸ" бһ¬ "бһҳбһ·бһ“бһ–бһ·бһҸ" (бһҒбһ»бһҹбһ–бһёбһңбһ·бһ’бһёбһҖбһ¶бһҡ "бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ¬бһҳбһ·бһ“" бһҖбҹ’бһ“бһ»бһ„ GAN бһ”бҹ’бһҡбһ–бҹғбһҺбһё)бҹ”

бһҹбһјбһҳбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё VQGAN бһ“бҹ…бһҸбҹҶбһ”бһ“бҹӢбһңбҹҒбһ” [Taming Transformers](https://compvis.github.io/taming-transformers/)бҹ”

бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҡбһңбһ¶бһ„ VQGAN бһ“бһ·бһ„ GAN бһ”бҹ’бһҡбһ–бҹғбһҺбһёбһӮбһәбһҗбһ¶ GANбһ”бҹ’бһҡбһ–бҹғбһҺбһёбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһӣбҹ’бһўбһ–бһёбһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бһјбһӣбһҺбһ¶бһЎбһҫбһҷ бһҒбһҺбҹҲбһҠбҹӮбһӣ VQGAN бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҳбһ·бһ“бһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ…бҹҶбһ“бһҪбһ“ бҹ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮ бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҠбһ№бһҖбһ“бһ¶бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ CLIPбҹ”

![VQGAN+CLIP Architecture](../../../../../translated_images/km/vqgan.5027fe05051dfa31.webp)

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ“бһ№бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһң бһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһңбҹүбһ·бһ…бһ‘бҹҗбһҡбһ‘бҹ…бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһ“бһҸбһ¶бһҳ VQGAN бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ CLIP бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ бһ‘бҹ…бһҸбһ¶бһҳбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбһҳбҹ’бһҡбһјбһңбҹ” бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһ“бҹҒбҹҮ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ back propagation бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһҗбҹҒбһҡinputs vectorбҹ”
