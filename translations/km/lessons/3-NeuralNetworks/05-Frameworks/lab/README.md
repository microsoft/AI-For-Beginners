# ការបែងចែកចំណាត់ថ្នាក់ជាមួយ PyTorch/TensorFlow

ភារកិច្ចមន្ទីរ​បច្ចេកវិទ្យា​ពី [កម្មវិធីសិក្សា AI សម្រាប់អ្នកចុងក្រោយ](https://github.com/microsoft/ai-for-beginners)។

## ភារកិច្ច

ដោះស្រាយបញ្ហាបែងចែកចំណាត់ថ្នាក់ពីរដោយប្រើបណ្តាញបណ្តុះបណ្តាលគ្រប់ជាន់តែមួយ និងច្រើនជាន់ ដោយប្រើ PyTorch ឬ TensorFlow៖

1. បញ្ហា **[ការបែងចែកចំណាត់ថ្នាក់ដើមឡូត្រូវ](https://en.wikipedia.org/wiki/Iris_flower_data_set)** - ជាឧទាហរណ៍នៃបញ្ហាជាមួយទិន្នន័យបែបតារាង ដែលអាចដោះស្រាយដោយមធ្យោបាយ ការបង្រៀនម៉ាស៊ីនបែបបុរាណ។ គោលបំណងរបស់អ្នកគឺត្រូវបែងចែកដើមឡូត្រូវជា 3 ថ្នាក់ ដោយផ្អែកលើប៉ារ៉ាម៉ែត្រជាលេខ 4។
1. បញ្ហាការបែងចែកចំណាត់ថ្នាក់លេខសរសេរដៃ **MNIST** ដែលយើងបានឃើញមុនរួច។

សាកល្បងស្ថាបត្យកម្មបណ្តាញផ្សេងៗ ដើម្បីទទួលបានភាពត្រឹមត្រូវល្អបំផុតដែលអ្នកអាចទទួលបាន។
