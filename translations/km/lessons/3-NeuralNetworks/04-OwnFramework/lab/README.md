# ការបែងចែកប្រភេទ MNIST ជាមួយស៊ុមរចនាសម្ព័ន្ធផ្ទាល់ខ្លួន

ការសិក្សាក្នុងមន្ទីរពិសោធន៍ពី [មេរៀន AI សម្រាប់អ្នកចាប់ផ្តើម](https://github.com/microsoft/ai-for-beginners)។

## ភារកិច្ច

ដោះស្រាយបញ្ហាបែងចែកលេខសរសេរដោយដៃ MNIST ដោយប្រើ perceptron មាន 1, 2 និង 3 ស្រទាប់។ ប្រើស៊ុមបណ្តាញសរសៃប្រសាទដែលយើងបានបង្កើតក្នុងមេរៀន។

## កំណត់ត្រា Notebook

ចាប់ផ្តើមមន្ទីរពិសោធន៍ដោយបើក [MyFW_MNIST.ipynb](MyFW_MNIST.ipynb)

## សំណួរ

ជាលទ្ធផលនៃមន្ទីរពិសោធន៍នេះ សូមព្យាយាមឆ្លើយសំណួរខាងក្រោម៖

- តើមុខងារបង្កើតសកម្មភាពរវាងស្រទាប់ មានឥទ្ធិពលដល់កម្រិតសមត្ថភាពបណ្តាញទេរឺ?
- តើយើងត្រូវការបណ្តាញមាន 2 ឬ 3 ស្រទាប់សម្រាប់ភារកិច្ចនេះទេ?
- តើអ្នកបានប្រឈមមុខនឹងបញ្ហាណាមួយក្នុងការបណ្តុះបណ្តាលបណ្តាញទេ? ជាពិសេសនៅពេលជាន់ស្រទាប់កើនឡើង។
- តើទំងន់នៃបណ្តាញមានអាកប្បកិរិយាយ៉ាងដូចម្តេចក្នុងអំឡុងការបណ្តុះបណ្តាល? អ្នកអាចគូសតម្លៃអប្បបរមា នៃទំងន់ទល់នឹង epoch ដើម្បីយល់ពីទំនាក់ទំនងនេះ។
