## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡ learns

**[бһ‘бһёбһҸбһ¶бҹҶбһ„бһ•бҹӮбһ“бһҖбһ¶бһҡвҖӢвҖӢбһ“бҹғбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶](http://soshnikov.com/courses/ai-for-beginners/mindmap.html)**

бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһўбҹҶбһ–бһёбҹ–

* бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҡбһ”бһҹбҹӢбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бһҠбһјбһ…бһҮбһ¶ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһүбҹ’бһүбһ¶бһҹбһ¶бһҡ "бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ…бһ¶бһҹбҹӢ" бһҮбһ¶бһҳбһҪбһҷ **бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһҠбҹҶбһҺбһ№бһ„** бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ бҹҒбһҸбһ»бһ•бһӣ ([GOFAI](https://en.wikipedia.org/wiki/Symbolic_artificial_intelligence))бҹ”
* **бһ”бһҺбҹ’бһҸбһ¶бһүбһ”бҹ’бһҡбһҹбһ¶бһ‘** бһ“бһ·бһ„ **бһҖбһ¶бһҡбһҡбҹҖбһ“бһҮбҹ’бһҡбҹ…**, бһҠбҹӮбһӣбһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғ AI бһ‘бҹҶбһ“бһҫбһ”бҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҡбһҝбһ„бһ–бһёбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҹбҹҠбҹҒбһҡбһёбһ–бһёбһҡбһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ”бҹҶбһ•бһ»бһҸ - [TensorFlow](http://Tensorflow.org) бһ“бһ·бһ„ [PyTorch](http://pytorch.org)бҹ”
* **бһҹбҹҶбһҺбһ„бҹӢбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бһ‘**бҹ” бһҷбһҫбһ„бһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҗбҹ’бһҳбһёбҹ— бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһўбһ¶бһ…бһҳбһ¶бһ“бһҒбҹ’бһңбҹҮбһҒбһ¶бһҸбһҒбҹ’бһӣбҹҮбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһӮбҹҶбһҡбһјбһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸбҹ”
* бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ AI бһҠбҹӮбһӣбһҳбһ·бһ“бһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҒбҹ’бһӣбҹҮ бһҠбһјбһ…бһҮбһ¶ **бһўбһ¶бһӣбҹӢбһ бҹ’бһӮбһ¶бһҡбһёбһ’аё¶бһҳбһҮбҹӮбһҳбһ”бҹғбһҸбһ„** бһ“бһ·бһ„ **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бһ¶бһӮбһёбһ…бҹ’бһҡбһҫбһ“**бҹ”
