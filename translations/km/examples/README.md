## рҹ“– бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҷбһҫбһ„бһҹбһјбһҳбһҺбҹӮбһ“бһ¶бҹҶбһІбҹ’бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбҹ–

1. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ "Hello AI World"** - бһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҳбҹүбһјбһҸ
2. **бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһҺбҹ’бһҠбһ¶бһүбһҹбһҡбһҹбҹғбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҹбһ¶бһҳбһүбҹ’бһү** - бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бһҺбҹ’бһҠбһ¶бһүбһ”бҹ’бһҡбһҹбһ¶бһ‘бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
3. **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбһјбһ”бһ—бһ¶бһ–** - бһҳбһҫбһӣ AI бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһ·бһҸ
4. **бһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһўбһҸбҹ’бһҗбһ”бһ‘** - бһҹбһ·бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ—бһ¶бһҹбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбһ’бһҳбҹ’бһҳбһҸбһ¶

## рҹ’Ў бһӮбһ“бҹ’бһӣбһ№бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

- **бһўбһ¶бһ“бһҳбһҸбһ·бһҖбһјбһҠбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“** - бһ–бһҪбһҖбһңбһ¶бһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бһёбһҳбһҪбһҷбҹ—бһ’бҹ’бһңбһҫ
- **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„!** - бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бһҳбҹ’бһӣбҹӮбһ„бһҸбһҳбҹ’бһӣбҹғ бһ бһҫбһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„
- **бһҖбһ»бҹҶбһ”бһ¶бһҡбһҳбҹ’бһ—бһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ** - бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶
- **бһҹбһҪбһҡбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘** - бһ”бҹ’бһҡбһҫ [бһҖбҹ’бһҡбһ»бһҳбһҹбһ“бҹ’бһ‘бһ“бһ¶](https://github.com/microsoft/AI-For-Beginners/discussions)
