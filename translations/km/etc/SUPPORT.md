# គាំទ្រ

## របៀបដាក់បញ្ហា និងទទួលបានជំនួយ  

គម្រោងនេះប្រើ GitHub Issues ដើម្បីតាមដានកំហុស និងសំណើមុខងារ។ សូមស្វែងរកបញ្ហាដែលមានស្រាប់ មុនពេលដាក់បញ្ហាថ្មីដើម្បីជៀសវាងការរួមបញ្ចូល។ សម្រាប់បញ្ហាថ្មី ដាក់កំហុសរបស់អ្នក ឬសំណើមុខងារជាបញ្ហាថ្មី។

សម្រាប់ជំនួយ និងសំណួរទាក់ទងនឹងការប្រើគម្រោងនេះ សូមប្រើកន្លែងសម្តែងមតិយោបល់។
